## 摘要
分割是计算病理学中的一项关键任务，因为它能识别出受疾病或异常增生影响的区域，对于诊断和治疗至关重要。然而，获取高质量的像素级监督分割数据需要经验丰富的病理学家投入大量工作量，限制了深度学习的应用。为应对这一挑战，放宽标注条件，仅使用图像级分类标签可以利用更多数据并拓展更多应用场景。一种常见方法是利用类激活映射（Class Activation Map, CAM）在仅有图像级标签的情况下生成伪像素级标注，用于语义分割。然而，此方法未能充分挖掘病理图像的本质特征，因此只能识别出有限的小区域，不足以生成有效的伪掩膜。本文提出了一种受课程学习（curriculum learning）启发的基于混洗反馈学习（shuffle-based feedback learning）的新方法，以生成更高质量的伪语义分割掩膜。具体而言，我们在病理图像的patch级别进行混洗，模型根据先前学习过程中的反馈自适应地调整混洗策略。实验结果表明，所提出的方法在三个不同的数据集上均优于现有最先进的方法。

## Abstract
Segmentation is a critical task in computational pathology, as it identifies areas affected by disease or abnormal growth and is essential for diagnosis and treatment. However, acquiring high-quality pixel-level supervised segmentation data requires significant workload demands from experienced pathologists, limiting the application of deep learning. To overcome this challenge, relaxing the label conditions to image-level classification labels allows for more data to be used and more scenarios to be enabled. One approach is to leverage Class Activation Map (CAM) to generate pseudo pixel-level annotations for semantic segmentation with only image-level labels. However, this method fails to thoroughly explore the essential characteristics of pathology images, thus identifying only small areas that are insufficient for pseudo masking. In this paper, we propose a novel shuffle-based feedback learning method inspired by curriculum learning to generate higher-quality pseudo-semantic segmentation masks. Specifically, we perform patch level shuffle of pathology images, with the model adaptively adjusting the shuffle strategy based on feedback from previous learning. Experimental results demonstrate that our proposed approach outperforms state-of-the-arts on three different datasets.

## 二、方法论：SegMix 框架与核心思想

### 1. 核心思路
- **整体理念**：  
  SegMix 构建一个**基于混洗反馈学习（shuffle-based feedback learning）**的框架，灵感源自“课程学习（curriculum learning）”。
  - 通过在图像的 patch 层级进行混洗；
  - 模型自适应调整混洗比例与 patch 尺寸；
  - 实现从粗到细的渐进式多尺度学习。

### 2. 关键模块与结构
SegMix 主体由三个核心组件组成：
1. **反馈学习模块（Feedback Learning Module）**
   - 包含两个调度器（scheduler）：
     - *Patch size scheduler*：设定一个递减的 patch 尺寸序列 `[pn, pn−1, ..., p1]`（从大到小）。
     - *Shuffle ratio scheduler*：设定混洗比例增长函数 `fi+1 = fi * α`，在训练时根据反馈动态调整。
   - 根据当前训练损失 `l` 与阈值 `T` 判断：
     ```
     若 l < T：
       patch 尺寸减小，shuffle 比例增加；
     否则：
       保持当前设置；
     随训练进行，T逐渐降低。
     ```
   - 实现渐进式学习，从组织级到细胞级逐步细化。

2. **Patch 混洗模块（Shuffle Patches）**
   - 将病理图像拆分为大小为 `p×p×C` 的 patch。
   - 按混洗比例 `f` 在同一批次（batch）内选择部分“关联 patch”进行*就地混洗（in-place shuffle)*。
   - 混合后的图像定义为：
     ```
     Pmix,i = M ⊙ PF,i + (1−M) ⊙ PR,i
     ```
     其中 `M` 为二值掩膜决定混洗的区域，`F` 为固定 patch，`R` 为来自其他样本的混洗 patch。
   - 这一设计促进了**跨样本实例关系**和**软标签生成**。

3. **像素相关模块（Pixel Correlation Module, PCM）**
   - 基于 SEAM [17] 改进的像素相似性增强模块。
   - 计算邻近像素的余弦相似度：
     ```
     f(xi, xj) = σ(θ(xi)^T θ(xj) / ||θ(xi)|| ||θ(xj)||)
     ```
   - 使用加权和更新原始 CAM，以提升预测一致性与空间连续性。

### 3. 流程概述
- **训练阶段**：
  1. 输入病理图像分块并混洗；
  2. 模型同时接收原始与混洗图进行“双分支”训练；
  3. 根据反馈调度器调整 patch 尺寸和混洗比；
  4. 生成伪标签并逐步优化分割掩膜。

## 六、主要结论与发现

- SegMix 相比基线：
  - 在 ROSE 数据集上：DSC 提升至 60.7%，IoU 达 50.2%；
  - 在 WBC 数据集上：DSC 提升至 41.5%，IoU 达 29.4%；  
  - 在 MARS 数据集上：也略优于对比方法。
- 反馈学习机制使模型能根据训练动态调整策略，提升性能约 6–7%。
- 混洗操作有效减少病理图像中的“扰动”信息，使模型更聚焦真实结构。
- 消融实验表明：
  - *Group* 与 *Back* 变体虽有效提升，但不如完整 SegMix；
  - 独立混洗方式更能捕获跨样本实例关系。
