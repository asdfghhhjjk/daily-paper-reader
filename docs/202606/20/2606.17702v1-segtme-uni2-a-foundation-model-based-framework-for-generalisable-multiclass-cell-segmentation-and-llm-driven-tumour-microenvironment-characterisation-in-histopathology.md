## 摘要
从常规H&E染色组织学图像中表征肿瘤微环境(TME)需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出了SEGTME-UNI2，一个统一框架来满足这些需求。其核心是UNI2-UPERHOVER，一个双头分割模型，将UNI2-H病理基础模型(ViT-Giant，在来自10万张切片的超过1亿张图块上预训练)与两个平行的UperNet解码器配对：一个用于六类语义分割，另一个用于水平-垂直梯度回归，从而实现基于分水岭的核实例分离。为了解决大型真实世界存储库中缺乏像素级标注的问题，UNI2-UPERHOVER采用三阶段渐进式伪标签课程。每个阶段训练一个全新模型(不进行权重迁移)，完全通过提高伪标签质量来驱动改进：阶段1：使用人工标注的PanNuke(7901张图像，189744个细胞核，0.25微米/像素)。阶段2：使用阶段1模型在271,711个TCGA-UT尺度0图块(0.5微米/像素)上生成经熵过滤的伪标签。阶段3：使用阶段2模型在全部1,608,060个TCGA-UT图块(涵盖6种分辨率尺度，0.5-1.0微米/像素)上生成伪标签。分割输出输入到一个结构化的TME特征提取流水线，计算20多个每图块的组成、形态、空间熵和细胞间距离指标。这些指标编码为JSON，并传递给微调后的NVIDIA BioNeMo GPT模型，生成临床可解释的TME叙述。在预留的PanNuke和TCGA-UT分区上的初步验证显示了框架的可行性和内部一致性。我们公开发布了带伪标签的TCGA-UT数据集和UNI2-UPERHOVER检查点，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SEGTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UPERHOVER, a dual-head segmentation model pairing the UNI2-H pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UPERHOVER undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UPERHOVER checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

- **核心问题**：从H&E染色组织病理图像中表征肿瘤微环境（TME）需要同时完成三项任务：（1）多类细胞分割（语义分割与实例分离）、（2）生物特征提取、（3）生成临床可解释的报告。现有方法要么只能分割（如Hover-Net、CellPose）而缺乏组织语义，要么只能语义分割（如UperNet）而无法分离接触的细胞核；同时缺乏大规模像素级标注，TCGA虽有海量数据但无人工标注。
- **整体含义**：本文提出一个统一的端到端框架 **SegTME-UNI2**，融合基础模型、伪标签课程和LLM驱动叙述生成，旨在弥合小规模高质量标注与大规模真实临床数据之间的鸿沟，实现可推广、可解释的TME自动化分析。

## 2. 方法论

- **核心思想**：利用病理基础模型UNI2-H作为共享编码器，设计双头UperNet解码器同时输出语义分割和HV梯度图（用于分水岭实例分离）；通过三阶段渐进伪标签课程将PanNuke的人工标注知识逐步扩展到160万张TCGA-UT图块（无新增人工标注）；最后将分割结果转化为结构化TME特征，并由BioNeMo GPT模型生成临床叙述。

- **关键技术细节**：
  - **UNI2-UperHoVer架构**：
    - 共享骨干：UNI2-H（ViT-Giant，预训练于>1亿图块，来自10万张切片、40+癌症类型）。
    - 四尺度特征金字塔：从Transformer块5、11、17、23提取特征，经1×1卷积投影到FPN通道数（256/512/1024/2048）。
    - 双头UperNet解码器：语义头（6类：背景、肿瘤、炎症、结缔、死细胞、上皮）和HV回归头（2通道水平/垂直梯度）。两者参数不共享，分别独立优化。
  - **损失函数**：
    - 语义损失：交叉熵，忽略ignore_index=255。
    - HV回归损失：MSE + 梯度匹配项（空间梯度误差乘以权重2）。
    - 总损失：L_sem + λ L_hv（λ=1）。
  - **动态HV目标合成**：从语义掩码通过连通分量分析、质心计算并归一化到[-1,1]范围，无需实例级标注即可生成HV目标。这使得伪标签阶段2、3能产生语义和HV目标。
  - **三阶段渐进伪标签课程**（不进行权重迁移，每次从头初始化）：
    - 第1阶段：在PanNuke（7901张，0.25 μm/pixel）上训练模型M1。
    - 第2阶段：M1在TCGA-UT scale-0（271,711张，0.5 μm/pixel）上推理，按熵过滤保留前70%置信图块；从头训练M2。
    - 第3阶段：M2在所有尺度（1,608,060张，0.5-1.0 μm/pixel）上推理+熵过滤；从头训练M3（最终模型）。
  - **TME特征提取与叙述生成**：
    - 特征：20+个每图块指标（细胞计数、比例、平均面积、肿瘤-间质分数、免疫浸润分数、免疫-肿瘤最小/平均距离、空间熵等）。
    - 规则翻译：依据阈值将数值特征映射为6个表型轴（组成、免疫、间质、空间、结构、形态）的标签和扁平词汇表。
    - BioNeMo GPT微调：以结构化JSON为输入，微调LLM生成临床叙述。损失为交叉熵，仅计算叙述部分。

## 6. 主要结论与发现

- 提出的三阶段渐进伪标签课程有效：无需额外人工标注，就能将分割能力从PanNuke扩展到160万张TCGA-UT多尺度图块。各阶段起始mIoU依次提高（0.41→0.47→0.53），表明伪标签质量逐阶段改善。

- 公开了最大规模核级标注TCGA数据集（1,608,060张伪标注图块）及三个模型检查点，为后续研究提供基础。

## 7. 优点

- **方法创新性**：
  - 首次将病理基础模型UNI2-H与双头UperNet结合，同时实现语义分割和实例分离。
  - 提出动态HV目标合成，仅需语义掩码即可训练实例分割，消除了实例级标注瓶颈。
  - 三阶段渐进课程（不迁移权重）新颖地利用熵过滤逐步扩大域，避免单步自训练的误差累积。
- **工程完整性**：端到端流水线从像素到临床叙事，包含MPP归一化、重叠分块、分水岭后处理、特征提取、规则翻译、LLM生成，模块化设计便于扩展。
- **实用性**：TME报告基于聚合统计特征而非单细胞精度，对分割误差有内在鲁棒性，更适合临床真实场景。公开数据集和模型权重促进可重复研究。
