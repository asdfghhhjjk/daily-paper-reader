## 摘要
从常规H&E染色组织学图像中表征肿瘤微环境（TME）需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出了SEGTME-UNI2，一个满足这些需求的统一框架。其核心是UNI2-UPERHOVER，一个双头分割模型，将UNI2-H病理学基础模型（ViT-Giant，在来自10万张切片的超过1亿张图块上预训练）与两个平行的UperNet解码器配对：一个用于六类语义分割，另一个用于水平-垂直梯度回归，从而实现基于分水岭的细胞核实例分离。为了解决大型真实世界数据集中像素级标注的缺乏，UNI2-UPERHOVER采用了三阶段渐进式伪标签课程。每个阶段训练一个全新的模型，不进行权重迁移，仅通过提高伪标签质量来驱动改进：阶段1：使用人工标注的PanNuke（7,901张图像，189,744个细胞核，0.25微米/像素）。阶段2：使用阶段1模型在271,711张TCGA-UT尺度0图块（0.5微米/像素）上生成的熵过滤伪标签。阶段3：使用阶段2模型在全部1,608,060张TCGA-UT图块上（涵盖0.5-1.0微米/像素的六个分辨率尺度）生成的伪标签。分割输出输入到结构化的TME特征提取流水线，计算每个图块的20多个组成、形态、空间熵和细胞间距离指标。这些指标编码为JSON，并传递给微调后的NVIDIA BioNeMo GPT模型，以生成临床可解释的TME叙述。在留出的PanNuke和TCGA-UT分区上的初步验证证明了框架的可行性和内部一致性。伪标签化的TCGA-UT数据集和UNI2-UPERHOVER检查点已公开发布，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SEGTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UPERHOVER, a dual-head segmentation model pairing the UNI2-H pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UPERHOVER undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UPERHOVER checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 核心思想
- **UNI2-UperHoVer双头分割架构**：以UNI2-H（ViT-Giant，在>1亿张图块上预训练）作为共享骨干，连接两个独立的UperNet解码器头——一个用于6类语义分割（Neoplastic, Inflammatory, Connective, Dead, Non-neoplastic Epithelial, Background），另一个用于水平-垂直（HV）梯度回归，从而通过分水岭算法实现核实例分离。双头设计分离了离散分类和连续回归目标，避免参数共享带来的冲突。
- **三阶段渐进式伪标签课程**：不使用任何权重迁移，每阶段从头训练模型，仅通过前一阶段生成的伪标签质量的提升来驱动。第一阶段用人工标注的PanNuke训练；第二阶段用第一阶段模型在TCGA-UT尺度0（0.5 μm/pixel）上推理并应用熵阈值过滤高置信度伪标签，再用这些伪标签从头训练第二阶段模型；第三阶段用第二阶段模型在所有六个分辨率尺度（0.5 ~ 1.0 μm/pixel）共1,608,060张图块上生成熵过滤伪标签，从头训练最终模型。
- **结构化TME特征提取与BioNeMo叙述生成**：分割后流水线计算每个图块的20多个定量指标（细胞计数、比例、平均面积、肿瘤-间质分数、免疫浸润分数、基于KD树的免疫-肿瘤距离、基于k近邻香农熵的空间混合指数等），转换为结构化JSON，再通过规则翻译为生物学标记（六个表型轴：组成、免疫状态、间质状态、空间相互作用、结构、形态），最后用微调的NVIDIA BioNeMo GPT模型生成临床风格的叙述文本。

### 2.2 关键技术细节
- **UNI2-H多尺度特征金字塔**：从ViT-Giant的24个Transformer块中选取块5、11、17、23（分别对应约25%、50%、75%、100%深度），提取四个尺度的特征，通过1×1卷积投影到标准FPN通道数（256,512,1024,2048），再双线性插值对齐到对应步长。
- **双UperNet解码器**：每个解码器包含金字塔池化模块（PPM）和FPN融合步骤，语义头输出6通道logits，HV回归头输出2通道梯度图（无激活函数）。HV回归损失由像素级MSE（公式3）和梯度匹配项MSGE（公式4，权重2）组成，总损失为语义交叉熵+λ·HV损失（λ=1.0）。
- **动态HV目标合成**：由于伪标签仅为语义掩码，无法提供实例级多边形标注。该方法通过对语义掩码进行连通组件分析，计算每个组件的质心，然后为每个像素分配相对于质心的归一化距离（裁剪到[-1,1]范围）作为HV目标，从而在纯语义标注上生成HV训练目标。
- **三阶段课程细节**：阶段1：在PanNuke上训练250 epochs；阶段2：用阶段1模型在TCGA-UT尺度0的271,711张图块上推理，根据每个图块前景像素平均熵（先验）选取前70%最确定图块（熵阈值设为第70百分位数），并用这些伪标签从头训练250 epochs；阶段3：用阶段2模型在所有1,608,060张图块上生成熵过滤伪标签，从头训练100 epochs（由于数据量巨大，实际梯度步数最多）。
- **TME特征提取**：使用`skimage.measure.regionprops`提取核质心和面积，计算每类计数、比例、平均面积、肿瘤-间质分数、免疫浸润分数。空间分析：在肿瘤细胞质心上建KD树，查询每个免疫细胞到最近肿瘤细胞的距离（最小、平均、50像素阈值内接触数）；在所有细胞质心上建KD树，为每个细胞检索最近的6个邻居，计算各细胞邻居类别的香农熵并取平均值。
- **生物学标记翻译**：基于固定规则阈值（如免疫浸润>30%→"Immune Hot"，免疫-肿瘤平均距离>60 px→"Excluded"，熵>0.70→"Highly Mixed"等）产生6个表型标签和11个扁平生物学标记，用于Prompt构建。

### 2.3 公式或算法流程
- 总损失函数：
  - 语义交叉熵：$\mathcal{L}_{\text{sem}} = -\frac{1}{|\Omega_v|}\sum_{p\in\Omega_v}\log \hat{p}_{y_p}(p)$（忽略索引255）
  - HV MSE：$\mathcal{L}_{\text{MSE}} = \frac{1}{|\Omega_{\text{fg}}|}\|(\hat{\mathbf{M}}-\mathbf{T})\odot\mathbf{M}_{\text{fg}}\|_F^2$
  - 梯度匹配：$\mathcal{L}_{\text{MSGE}} = \frac{1}{|\Omega_{\text{fg}}|}\left[\|(\nabla_x\hat{M}_0-\nabla_x T_0)\odot M_{\text{fg}}\|_F^2 + \|(\nabla_y\hat{M}_1-\nabla_y T_1)\odot M_{\text{fg}}\|_F^2\right]$
  - 总HV：$\mathcal{L}_{\text{hv}} = \mathcal{L}_{\text{MSE}} + 2\mathcal{L}_{\text{MSGE}}$
  - 总目标：$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{sem}} + \lambda \mathcal{L}_{\text{hv}}, \lambda=1.0$
- 伪标签熵过滤：$H(p) = -\sum_{c=0}^5 \hat{p}_c(p)\log\hat{p}_c(p)$，图块置信度得分 $\bar{H}$ 为前景像素平均熵，仅保留 $\bar{H}<\tau$（$\tau$为第70百分位数）的图块。

## 7. 优点

- **架构创新**：双头UperNet设计（语义头+HV头）共享UNI2-H骨干，通过动态HV目标合成解决了从纯语义标注学习实例分离的问题，避免了对实例级标注的依赖。
- **可扩展的伪标签课程**：三阶段渐进式课程依次引入尺度0（最接近PanNuke分辨率），然后扩展到所有尺度，给模型每次巩固机会，避免单步自训练中的错误累积。不继承权重，纯粹通过伪标签质量提升驱动进步。
- **多尺度适应性**：选择UperNet解码器是因为其PPM+FPN结构在处理从0.25到1.0 μm/pixel的宽尺度范围时优于点式解码器（UNet、SegFormer等），这是解决分辨率迁移的关键。
- **可解释性整合**：通过规则翻译将定量特征转化为生物学标记，再通过BioNeMo SFT生成叙述，叙述可追溯到具体数值，极大降低幻觉风险。且生成质量对单细胞分割精度不敏感（依赖汇总统计），增强了鲁棒性。
- **开放贡献**：公开了伪标签化的TCGA-UT数据集（1.6M图块）和三个阶段的模型检查点，提供pip安装包，有利于社区复现和大规模研究。

## 8. 不足与局限

- **实验验证不足**：目前仅在PanNuke测试集上评估了mIoU（与单头UperNet对比），缺乏与其他SOTA方法（HoVer-Net、CellPose、StarDist）在标准公共基准（MoNuSeg、CoNSeP、CryoNuSeg）上的定量比较；缺乏消融实验（HV头贡献、熵阈值敏感性、课程阶段必要性、解码器选择等）；缺乏统计显著性测试（无置信区间、无多种子实验）；未对TCGA-UT伪标签进行人工金标准验证（仅计划200张图块）；未进行任何下游临床结局关联分析。
- **伪标签质量未验证**：虽然各阶段模型在TCGA-UT验证mIoU递增，但该验证集也是伪标签，并非人工标注，存在自我验证循环风险。真实准确度未知。
- **部署局限**：动态HV目标合成对同类别相邻核的近似处理可能引入误差；分水岭分割在密集重叠核区域可能失败；瓦片级推理中的最后写入HV图拼接可能产生梯度不连续性。
- **规则阈值偏倚**：生物学标记翻译使用固定规则阈值（如30%、60 px等），这些阈值来自文献但未必泛化到所有癌症类型和组织场景。作者计划后续用结果监督分类器替代。
- **推理速度**：UNI2-H为ViT-Giant（>600M参数），加上双解码器，推理速度可能较慢（文中未报告）。
- **叙述质量未量化**：未使用BLEU、ROUGE、病理学家间一致性等指标评估BioNeMo生成叙述的保真度和临床实用性。
