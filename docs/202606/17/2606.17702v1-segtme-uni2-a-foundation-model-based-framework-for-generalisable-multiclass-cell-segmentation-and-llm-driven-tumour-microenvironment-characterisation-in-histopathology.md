## 摘要
从常规的H&E染色组织学图像中表征肿瘤微环境（TME）需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出了SEGTME-UNI2，一个统一框架以满足这些需求。其核心是UNI2-UPERHOVER，一个双头分割模型，将UNI2-H病理基础模型（ViT-Giant，在来自10万张切片的>1亿张图块上预训练）与两个并行的UperNet解码器配对：一个用于六类语义分割，另一个用于水平-垂直梯度回归以实现基于分水岭的细胞核实例分离。为了解决大型真实世界数据集中缺乏像素级标注的问题，UNI2-UPERHOVER经历了三阶段渐进式伪标签课程。每个阶段都训练一个全新模型（无权重迁移），完全通过提高伪标签质量来推动改进：第一阶段：使用人工标注的PanNuke（7,901张图像，189,744个细胞核，0.25微米/像素）。第二阶段：对271,711个TCGA-UT比例0图块（0.5微米/像素），使用第一阶段模型的熵过滤伪标签。第三阶段：对全部1,608,060个TCGA-UT图块（覆盖0.5-1.0微米/像素的六种分辨率尺度），使用第二阶段模型的伪标签。分割输出馈入一个结构化的TME特征提取流水线，计算每个图块的20多种成分、形态、空间熵和细胞间距离指标。这些特征被编码为JSON并传入微调后的NVIDIA BioNeMo GPT模型，以生成临床可解释的TME描述。在预留的PanNuke和TCGA-UT分区上的初步验证表明了框架的可行性和内部一致性。公开了带伪标签的TCGA-UT数据集和UNI2-UPERHOVER检查点，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SEGTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UPERHOVER, a dual-head segmentation model pairing the UNI2-H pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UPERHOVER undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UPERHOVER checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

### 2. 论文提出的方法论

- **核心思想**：设计一个双头分割模型（UNI2-UPERHOVER），以共享的 UNI2-H ViT-Giant 编码器提取多尺度特征，再通过两个独立的 UperNet 解码器分别输出六类语义分割图和水平‑垂直（HV）梯度图。语义分割图用于细胞类型分类，HV 梯度图通过分水岭算法实现实例分离。通过三阶段渐进式伪标签课程，利用一个阶段模型生成的熵过滤伪标签训练下一阶段模型（无权重继承），从而逐步将监督信号从高分辨率 PanNuke（0.25 μm/pixel）扩展到 TCGA-UT 的六种分辨率尺度（0.5–1.0 μm/pixel）。最后，从分割结果中提取 20+ 种 TME 指标（成分、空间熵、免疫-肿瘤距离等），编码为 JSON 后驱动微调的 BioNeMo GPT 模型生成临床叙述。
- **关键技术细节**：
    - **UNI2-UPERHOVER 结构**：从 UNI2-H 第 5、11、17、23 个 Transformer 块提取四尺度特征（1/4、1/8、1/16、1/32 步长），通过 1×1 卷积投影到标准 FPN 通道数（256/512/1024/2048）。两个解码器均采用 PPM + FPN 融合结构，参数不共享。
    - **损失函数**：语义交叉熵损失（忽略 ignore_index=255） + HV 回归损失（包含像素级 MSE 项和梯度匹配 MSE 项，总权重 λ=1.0）。梯度匹配项通过 Sobel 算子计算 HV 图的空间梯度，增强边界位置的预测精度。
    - **动态 HV 目标合成**：仅需语义遮罩即可合成 HV 目标。通过连通组件分析获取细胞核质心，计算每个像素相对质心的归一化向量（clip 到 [-1,1]），无需实例标注。
    - **三阶段课程**：
        - **Stage 1**：在 PanNuke 上全监督训练 √1。
        - **Stage 2**：√1 对 TCGA-UT Scale 0（0.5 μm/pixel）的 271,711 个图块进行推理，计算每像素熵并按图块平均熵进行阈值过滤（保留 Top 70% 置信图块），用过滤后的伪标签从头训练 √2。
        - **Stage 3**：√2 对全部 1,608,060 个图块（Scales 0–5）进行推理，同样过滤后从头训练 √3 作为最终模型。
    - **TME 特征提取流水线**：从实例遮罩中计算每个类别的细胞计数、比例、平均面积；生成肿瘤‑间质分数、免疫浸润分数；通过 cKDTree 计算免疫细胞到肿瘤细胞的最近/平均距离、接触免疫细胞数量；计算每个细胞的局部 Shannon 熵（k=5 个最近邻）并取平均得到空间混合指数。
    - **生物标记翻译与 BioNeMo 叙述生成**：基于阈值规则（如免疫比例 >0.30 → "Immune Hot"）将数值特征转换为 6 个表型分类标签和 11 个扁平生物标记词。结构化 JSON（raw_data, interpretation, generated_tokens）作为提示，微调 NVIDIA BioNeMo GPT 模型以自回归方式生成临床叙述。

### 6. 论文的主要结论与发现

- **结论**：SEGTME-UNI2 框架能够实现从 H&E 图像到临床叙述的端到端 TME 表征。三阶段渐进式伪标签课程可以成功将 PanNuke 的监督信号扩展到百万级 TCGA-UT 图块，且每一阶段模型的初始 mIoU 逐步提高（Stage 1: 0.41 → Stage 2: 0.47 → Stage 3: 0.53），证明伪标签质量随阶段递增。最终模型 Stage 3 在 PanNuke 测试集上最佳 mIoU 达到 0.931（但注意这是基于 Stage 1 自身时的结果，Stage 2/3 在 PanNuke 上的 mIoU 未明确给出，但从图 7 看 Stage 3 的 PanNuke mIoU 曲线低于 Stage 1，这反映了跨域泛化下降，但仍在合理范围）。定性 HNSCC 案例显示，即使染色条件较差（如 Case D 淡粉色脱浆），基于统计聚合特征的 TME 表型分类仍保持临床一致性。
- **发现**：UperNet 解码器因其 PPM+FPN 的多尺度融合能力，更适合处理训练（0.25 μm/pixel）与推理（0.5–1.0 μm/pixel）的分辨率差异。动态 HV 目标合成使得伪标签课程可以在无实例标注的情况下进行。将 LLM 输出锚定在定量特征上可降低幻觉风险。

### 7. 优点

- **创新性**：将大规模病理基础模型（UNI2-H）与双头 UperNet 解码器结合，首次实现六类语义+HV 梯度的一体化分割输出；提出三阶段渐进式伪标签课程，利用熵过滤逐步扩大域覆盖，避免单步自训练的错误累积。
- **规模性**：生成了目前最大的带有核级伪标签的 TCGA 语料库（1,608,060 个图块），并公开发布，有助于下游空间生物学研究。
- **实用性**：设计了完整的 TME 特征提取流水线和基于阈值的生物标记翻译，使临床叙述可追溯到具体测量值；通过 BioNeMo SFT 将结构化数据转化为自然语言，降低临床使用门槛。
- **对分割缺陷的鲁棒性**：由于叙述基于聚合统计而非单细胞边界精度，即使分割质量因染色条件下降，整体表型分类仍保持合理（如定性案例 D）。

### 8. 不足与局限

- **实验覆盖不足**：
    - 缺乏与主流方法（HoVer-Net, CellPose, StarDist）在外部基准（MoNuSeg, CryoNuSeg, CoNSeP）上的定量对比。
    - 未进行消融实验（HV 头贡献、阶段数目、熵阈值等）。
    - 伪标签质量未与人类标注进行直接比对（计划用 200 张人工标注 TCGA-UT 图块）。
    - 未报告置信区间、p 值或多随机种子实验，统计严谨性不足。
- **验证偏差**：定性案例来自单一机构（UMCRI），且仅 4 例，代表性有限。TME 特征与临床预后的关联尚未建立（列为未来工作）。
- **技术局限**：
    - 动态 HV 目标合成在密度高或同类细胞相邻时存在近似误差。
    - 图块边界拼接时使用 last-writer 策略可能导致 HV 梯度不连续。
    - 生物标记阈值基于文献，可能不适用于所有癌种或染色协议，未通过数据驱动优化。
- **应用限制**：整个流水线依赖于 UNI2-H 基础模型（约 6 亿参数）和 BioNeMo GPT，推理成本较高；仅适用于 H&E 染色；需 MPP 归一化和 50% 重叠分块，处理效率有待评估。
