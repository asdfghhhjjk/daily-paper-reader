## 摘要
对组织病理图像的精确分析对于疾病诊断和治疗规划至关重要。将组织切片以千兆像素分辨率数字化的 Whole-slide images（WSI）是这一过程的基础，但在实现切片级预测时需要聚合成千上万的图块。Multiple Instance Learning（MIL）通过两阶段范式解决了这一挑战，将图块级嵌入与切片级预测解耦。然而，大多数现有方法在均质的欧几里得空间中隐式嵌入图块表示，忽视了病理组织的层次结构和区域异质性。这限制了当前模型捕获全局组织结构和细粒度细胞形态的能力。为了解决这一局限，我们引入了一种混合的双曲-欧几里得表示方法，在双几何空间中嵌入 WSI 特征，使模型能够互补地建模组织的层次结构与局部形态细节。在此基础上，我们提出了 BatMIL——一种利用双几何空间的 WSI 分类框架。为了建模数千图块之间的长程依赖关系，我们采用了结构化状态空间序列模型（S4）作为骨干，以线性计算复杂度对图块序列进行编码。此外，为了应对区域异质性，我们引入了一个块级 mixture-of-experts（MoE）模块，将图块分组为区域并动态路由到专门的子网络，从而在提高表示能力的同时减少冗余计算。在涵盖六种癌症类型的七个 WSI 数据集上进行的大量实验表明，BatMIL 在切片级分类任务中持续优于现有最先进的 MIL 方法。这些结果表明，具备几何感知的表示学习为下一代计算病理学提供了一个有前景的方向。

## Abstract
Accurate analysis of histopathological images is critical for disease diagnosis and treatment planning. Whole-slide images (WSIs), which digitize tissue specimens at gigapixel resolution, are fundamental to this process but require aggregating thousands of patches for slide-level predictions. Multiple Instance Learning (MIL) tackles this challenge with a two-stage paradigm, decoupling tile-level embedding and slide-level prediction. However, most existing methods implicitly embed patch representations in homogeneous Euclidean spaces, overlooking the hierarchical organization and regional heterogeneity of pathological tissues. This limits current models' ability to capture global tissue architecture and fine-grained cellular morphology. To address this limitation, we introduce a hybrid hyperbolic-Euclidean representation that embeds WSI features in dual geometric spaces, enabling complementary modeling of hierarchical tissue structures and local morphological details. Building on this formulation, we develop BatMIL, a WSI classification framework that leverages both geometric spaces. To model long-range dependencies among thousands of patches, we employ a structured state space sequence model (S4) backbone that encodes patch sequences with linear computational complexity. Furthermore, to account for regional heterogeneity, we introduce a chunk-level mixture-of-experts (MoE) module that groups patches into regions and dynamically routes them to specialized subnetworks, improving representational capacity while reducing redundant computation. Extensive experiments on seven WSI datasets spanning six cancer types demonstrate that BatMIL consistently outperforms state-of-the-art MIL approaches in slide-level classification tasks. These results indicate that geometry-aware representation learning offers a promising direction for next-generation computational pathology.

- **研究背景**  
  对组织病理学图像（Whole-Slide Images, WSI）的自动分析是精准诊断和治疗方案规划的重要基础，但由于切片级图像可达千兆像素规模，模型需在数千至万级图块间实现全局关联和分类，这带来了计算和表示上的挑战。  
- **问题与动机**  
  传统多实例学习（Multiple Instance Learning, MIL）方法将WSI划分为若干图块后在欧几里得空间中嵌入、再汇总做整体预测。  
  这些方法面临三大不足：
  1. 忽视组织的**层级结构（细胞→腺体→组织）**，单一欧几里得空间难以表达层次拓扑。  
  2. 缺乏对**长程空间依赖性**的有效建模。  
  3. 无法适应**区域异质性**（肿瘤、坏死、正常区域并存）。  
  因此，论文旨在提出一种能够同时建模层次结构、局部形态和空间依赖的新型表示方法。

## 二、方法论与技术框架

### 2.1 总体思想
提出 **BatMIL** 框架，即一个「几何感知 + 状态空间 + 动态专家混合」的多实例学习模型。  
它将 WSI 特征嵌入至**双几何空间（超双曲 + 欧几里得）**，结合结构化状态空间序列模型（S4）和混合专家（MoE）模块，以达到：
- 模型层级结构 → 超双曲空间；
- 局部形态细节 → 欧几里得空间；
- 全局依赖关系 → 状态空间模型；
- 区域异质性 → 动态专家路由。

### 2.2 关键模块

#### （1）图块预处理与嵌入
- 使用 224×224 的非重叠图块；
- 利用光照阈值与像素方差过滤背景；
- 采用 GigaPath 模型提取图块语义向量。

#### （2）结构化状态空间序列模块（S4）
- 用于编码超长图块序列，具有线性复杂度。  
  - 初始化方式：HiPPO算子（高阶多项式投影）。
  - 实现残差连接：
    \[
    x_{S4} = x + \text{DropPath}(S4(\text{LN}(x)))
    \]
  - 通过非方向性全局递归建模2D组织结构。

#### （3）混合专家路由（MoE）
- 建立 4 个专家网络（两层FFN，带GELU与Dropout）。  
- 动态选取每个图块的前2个高概率专家：
  \[
  x_{MoE} = x_{S4} + \text{DropPath}\left(\sum_{j\in I} w_j E_j(\text{LN}(x_{S4}))\right)
  \]
- 解决区域异质性问题，提高非线性表达力并减少计算冗余。

#### （4）几何混合表示（GHS）
- 在欧几里得空间和超双曲空间中联合建模。  
  - 超双曲距离定义：
    \[
    D_{hyp}(x, y) = 2\sqrt{c}\ \text{arctanh}(\sqrt{c - x \oplus_c y})
    \]
  - 融合策略采用**加权求和**（经验效果最佳）：
    \[
    z_{hybrid} = \alpha z_E + (1-\alpha)\log_c(z_H)
    \]
  其中 α 为可学习权重，用于平衡两种空间贡献。

### 2.3 模型输出
- 在图块维度上做最大池化，生成切片级表示；
- 再经多任务预测层输出分类或分级结果。

- **几何创新**：首创超双曲-欧几里得双空间的病理嵌入，有效整合层级与局部信息。  
- **结构优势**：S4状态空间模型捕获长程依赖并保持线性复杂度，适配超大图块序列。  
- **异质性建模**：MoE动态专家机制自适应区域差异，实现高效稀疏计算。

## 八、不足与局限

- **局部信号敏感度不足**：在极度稀疏的生物标志物任务（如HER2表达），性能略低于纯注意力模型。

- **存储与算力开销**：双几何嵌入和MoE增加一定显存需求，仍高于简化聚合器。  
- **未探讨跨模态扩展**：模型仍局限于WSI单模态，未检验与临床表型数据融合的效果。
