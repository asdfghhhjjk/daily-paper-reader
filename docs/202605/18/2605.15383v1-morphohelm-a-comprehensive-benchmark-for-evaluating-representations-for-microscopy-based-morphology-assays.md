## 摘要
显微镜图像包含了关于细胞对扰动反应的丰富信息，因此在药物筛选等应用中至关重要。为了量化图像，研究人员常使用表示提取方法，而近年来深度学习方法迅速发展。尽管衡量这些表示的质量至关重要，但当前的评估仍然分散，各模型在不同任务和数据集上使用自定义流程和指标进行评估，导致模型间难以进行公平比较。在此，我们提出了 MorphoHELM —— 一个用于评估 Cell Painting（最广泛使用的形态学表型分析实验）特征提取方法的综合性开放基准。MorphoHELM 整合了该领域的评估标准，对其进行了扩展和修正，使之更加稳健，并在迄今为止最广泛的方法范围内进行评估。该基准的一个显著特征是：每个任务都在不同程度的批次效应（或技术噪声）下进行评估，从而直接量化方法在噪声增加时检测生物信号能力的退化。上述特性使 MorphoHELM 能够揭示不同方法之间的权衡，我们展示了在某些类型生物信号上表现优异的模型在其他类型上可能较弱。我们进一步表明，目前没有任何现有模型能在所有设置中超过经典计算机视觉分析策略，而这些传统方法仍然是最强的通用表示方案。所有数据集、代码和评估工具均已在 https://github.com/microsoft/MorphoHELM 公开发布。

## Abstract
Microscopy images contain rich information about how cells respond to perturbations, making them essential to applications like drug screening. To quantify images, researchers often use representation extraction methods, and recent years have seen a proliferation of deep learning methods. While measuring the quality of these representations is essential, evaluation remains fragmented, with each proposed model evaluated on different tasks and datasets, using custom pipelines and metrics, making it difficult to fairly compare models. Here, we introduce MorphoHELM, a comprehensive open benchmark for evaluating feature extraction methods for Cell Painting, the most widely-used morphological profiling assay. MorphoHELM consolidates evaluation standards in the field, extends and corrects them to be more robust, and evaluates on the widest range of methods to date. A defining feature of the benchmark is that each task is evaluated at different degrees of batch effects (or technical noise), directly quantifying how the ability of methods to detect biological signal degrades as noise increases. Together, these properties enable MorphoHELM to detect trade-offs between methods, and we demonstrate that models that excel at certain kinds of biological signal are weaker at others. We show that no existing model outperforms classic computer vision analytic strategies across all settings, which remain the strongest general use-case representations. All datasets, code, and evaluation tools are publicly available at https://github.com/microsoft/MorphoHELM.

## 一、核心问题与研究动机

- **背景**：高通量显微镜成像（尤其是 Cell Painting 技术）能够捕捉细胞对化学或基因扰动的复杂反应。这类数据对于药物筛选、基因功能研究尤为重要。

  - 不同研究使用不同数据集与评估指标；
  - 自定义流程导致结果不可比较；
  - 对“批次效应”（batch effects，即实验噪声和设备差异）的鲁棒性评估不足。  
- **目标**：构建一个统一、公开、全面的评估基准，以系统比较不同特征提取和表示学习方法在显微镜形态学任务中的性能与鲁棒性。

## 二、方法论与技术框架

### 2.1 MorphoHELM 基准的核心思想

MorphoHELM（Holistic Evaluation of Learned Morphological Representations）旨在：
- 标准化形态学表示质量评估；
- 统一多种生物学信号任务；
- 在不同噪声/批次效应强度下量化各方法性能衰减。

### 2.2 三类评测任务

1. **MoA（Mechanism of Action）富集**  
   - 测试模型是否能基于形态表示将具有相同作用机制的化合物聚类。
2. **基因通路富集（Gene Pathway Enrichment）**  
   - 检测模型是否能在表征空间中聚合同一基因途径或蛋白复合体中的基因扰动。
3. **重复实验一致性（Replicate Retrieval Consistency）**  
   - 测试模型是否能准确检索相同扰动的实验重复样本。

### 2.3 批次效应分级设计（技术创新）

所有任务都在四个严格程度的“批次效应控制”下评估：
1. **NR（No Restriction）**：无约束；
2. **NSB（Not Same Batch）**：不同实验批次；
3. **NSS（Not Same Source）**：不同实验实验室或显微镜来源；
4. **NSL（Not Same Layout）**：不同板位布局。  
→ 此设计可量化模型在技术噪声增加时的表现下降，体现鲁棒性。

### 2.4 核心度量指标

- **富集类任务（MoA / Gene Pathway）**
  - 使用**几何平均优势比（Geometric Mean Odds Ratio，OR）**与**显著富集比例（Fraction Significant）**。
  - 引入**修正 Haldane–Anscombe 校正**，处理小样本导致的无限 OR 问题。
  - 利用随机置换检验计算显著性。
- **重复实验任务**
  - 局部度量：**kNN Recall@1**；
  - 全局度量：**Mean Average Precision (mAP)**；
  - 附加度量：**Negative Control mAP**（对照组区分度）。  
这些修改避免了此前基准在指标上偏向高维模型或特定结构的问题。

## 三、实验设计与 benchmark 规范

### 3.1 使用的数据集来源

来自 **Broad Bioimage Benchmark Collection (BBBC)** 与 **JUMP-CPG** 联合体，覆盖化学与基因扰动场景：

| 数据集 | 类型 | 是否跨机构复现 | 任务 |
|---------|------|---------------|------|
| BBBC036 | 化学 | 否 | MoA |
| cpg-MoA | 化学 | 是 | MoA |
| cpg-CRISPR | 基因敲除 | 否 | 基因通路富集 |
| cpg-target2 | 化学 | 是 | 重复检索 |
| cpg-compound | 化学（大规模） | 多机构 | 高难度重复检索 |

### 3.2 评估模型（八类）

1. **经典手工特征方法**：CellProfiler  
2. **自然图像迁移学习模型**：ResNet101（ImageNet预训练）  
3. **未训练的 ResNet（随机初始化）**：最低基线  
4. **自监督视觉模型**：DINOv2（ViT-B/14，含 CLS token 与 patch embedding 两种形式）  
5. **显微镜专用预训练模型**：
   - OpenPhenom（Cell Painting 上自监督掩码自动编码器）
   - SubCell（来自其他荧光显微镜数据）
   - CLOOME（图像–化学结构多模态模型）

### 3.3 预处理与标准化流程

保证各方法对比公平：
- 图像标准化：8-bit 转换、照明校正；
- 数据质量控制：去重频道、低细胞数滤除、异常批次剔除；
- 特征后处理：统一缩放、PCA 降维至 64 维、MAD 鲁棒标准化。

## 六、主要结论与发现

1. **无模型在所有任务上绝对占优**。  
   - DINOv2（自然图像自监督模型）整体性能最佳，但在部分任务中被显微镜模型或经典特征超越。
2. **批次效应显著影响性能**。  
   - “Not Same Source”场景下所有模型性能显著下降，说明跨实验室数据迁移仍是未解决问题。
3. **CellProfiler表征依然强劲**。  
   - 在“重复检索一致性”任务中经典手工特征优于所有深度学习模型，说明其鲁棒性高。
4. **显微镜预训练模型存在特化倾向**。  
   - CLOOME 针对基因通路任务表现好，但在化学富集任务上较弱；
   - SubCell 表现相反。
5. **自然图像模型表现广但不深**。  
   - 可检出更多但较浅的生物信号，而显微镜模型能捕捉少量但更细粒度的通路特征。
6. **后处理影响重大**：PCA 和归一化能显著提升性能，尤其对 CellProfiler 特征。
