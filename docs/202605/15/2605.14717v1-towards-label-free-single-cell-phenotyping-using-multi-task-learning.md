## 摘要
无标记单细胞成像为基于荧光的细胞分析提供了一种可扩展、非侵入性的替代方案，但要直接从明场形态中推断分子表型仍然具有挑战性。我们提出了一个统一的深度学习（DL）框架，可在无标记差分相衬（DPC）图像上同时执行白细胞（WBC）分类和连续蛋白表达回归。该模型采用一种混合结构，通过可学习的跨分支门控模块，将卷积的细粒度纹理特征与基于Transformer的全局表征融合，从而实现从DPC图像中稳健的形态-分子推断。为了支持后续的可解释性，我们进一步引入了大型语言模型（LLM），生成简洁且具有生物学依据的预测细胞状态摘要。在伯克利单细胞计算显微（BSCCM）和Blood Cells Image基准数据集上的实验表明，该方法表现优异，在BSCCM上实现了91.3%的白细胞分类准确率和CD16表达回归的0.72皮尔森相关系数。这些结果强调了无标记单细胞成像在经济高效的血液学分析中的潜力，使得无需荧光染色即可同时实现表型识别和定量生物标志物估计。源代码可访问：https://github.com/saqibnaziir/Single-Cell-Phenotyping。

## Abstract
Label-free single-cell imaging offers a scalable, non-invasive alternative to fluorescence-based cytometry, yet inferring molecular phenotypes directly from bright-field morphology remains challenging. We present a unified Deep Learning (DL) framework that jointly performs White Blood Cell (WBC) classification and continuous protein-expression regression from label-free Differential Phase Contrast (DPC) images. Our model employs a Hybrid architecture that fuses convolutional fine-grained texture features with transformer-based global representations through a learnable cross-branch gating module, enabling robust morpho-molecular inference from DPC images. To support downstream interpretability, we further incorporate a Large Language Model (LLM) that generates concise, biologically grounded summaries of the predicted cell states. Experiments on the Berkeley Single Cell Computational Microscopy (BSCCM) and Blood Cells Image benchmarks demonstrate strong performance, achieving a 91.3% WBC classification accuracy and a 0.72 Pearson correlation for CD16 expression regression on BSCCM. These results underscore the promise of label-free single-cell imaging for cost-effective hematological profiling, enabling simultaneous phenotype identification and quantitative biomarker estimation without fluorescent staining. The source code is available at https://github.com/saqibnaziir/Single-Cell-Phenotyping.

- **研究目标**：提出一种统一的深度学习框架，实现**同时的细胞类型分类与蛋白表达回归**，并提供可解释的生物学推理，从而实现“形态-分子联合推断”。

## 二、方法论：核心思想与技术实现

### 1. 总体框架
- 采用一个**多任务学习（Multi-task Learning）框架**，以 DPC 图像为输入，同时输出：
  - 离散的 **细胞类型标签（分类任务）**；
  - 连续的 **蛋白表达值（回归任务）**；  
- 引入一种 **混合式深度网络结构（Hybrid CNN–Transformer）**，结合局部纹理和全局空间信息。

### 2. 网络结构设计
- **输入**：四通道 DPC 图像（左、右、上、下方向照明）。  
  - 使用 **ECA Efficient Channel Attention** 模块自适应加权不同照明通道。  
- **CNN分支**：
  - 用于捕捉局部形态特征（如细胞膜、核纹理），采用 Inception 模块 + 残差连接；
  - 输出 196×192 的特征序列。  
- **Transformer（ViT）分支**：
  - 将 28×28 图像划分为 4×4 patch，共 49 个 patch；
  - 通过两层自注意力（4头）提取全局依赖；
  - 输出特征维度 50×128。  
- **跨模态融合（Cross-branch Fusion）**：
  - 将 CNN 全局平均池化结果与 ViT 的 CLS token 投影到 256维；
  - 使用 **可学习加权 softmax 融合**，得到综合特征 `hfused`。  

### 3. 任务自适应门控机制（Task-adaptive Gating）
- 在共享特征上建立两个专用通道（分类与回归），每个由两层 MLP 构成；
- 通过 **Sigmoid 激活的软门控层**实现跨任务信息共享：
  - 分类与回归特征可以相互借力，学习到细胞类型与蛋白表达之间的相关性。  

### 4. 预测头与损失函数
- **分类头**：三层 MLP + Softmax 输出；Dropout=0.4。  
- **回归头**：三层 MLP + 线性输出；高Dropout防止过拟合。  
- **总损失函数**：
  \[
  L_{\text{total}} = \lambda_{cls} L_{cls} + \lambda_{reg} L_{reg} + \lambda_{aux} L_{aux}
  \]
  - \(L_{cls}\)：Focal Loss（应对类别不平衡）；
  - \(L_{reg}\)：Smooth L1 + 负 PearsonCorr，用于连续值回归；
  - \(L_{aux}\)：特征一致性正则化。  

### 5. 可解释性模块
- 引入 **大型语言模型（LLM，Gemini 1.5 Pro）**：
  - 将模型定量预测结果（JSON格式）转化为模板化生物学描述；
  - 保证不产生推测性“幻觉”，仅基于模型预测；  
  - 不参与训练，仅做后处理解释层。
