## 摘要
近年来，计算机视觉技术取得了显著进展，并在医学领域找到了多种具有深远影响的应用。本文介绍了一种在巴氏涂片图像中检测 Bethesda 细胞的新框架，该框架开发用于与国际生物医学成像研讨会（ISBI）合作举办的 Riva Cytology Challenge 的 Track B。本研究重点在于提升用于细胞检测的计算机视觉模型的性能，并使用 mAP50-95 指标进行评估。我们提出的解决方案基于 YOLO 和 U-Net 架构的集成模型，随后通过利用重叠去除技术和二分类器的优化阶段进行细化。我们的框架在比赛中以 0.5909 的 mAP50-95 分数获得了第二名。实现与源代码可在以下仓库获取：github.com/martinamster/riva-trackb。

## Abstract
Computer vision techniques have advanced significantly in recent years, finding diverse and impactful applications within the medical field. In this paper, we introduce a new framework for the detection of Bethesda cells in Pap smear images, developed for Track B of the Riva Cytology Challenge held in association with the International Symposium on Biomedical Imaging (ISBI). This work focuses on enhancing computer vision models for cell detection, with performance evaluated using the mAP50-95 metric. We propose a solution based on an ensemble of YOLO and U-Net architectures, followed by a refinement stage utilizing overlap removal techniques and a binary classifier. Our framework achieved second place with a mAP50-95 score of 0.5909 in the competition. The implementation and source code are available at the following repository: github.com/martinamster/riva-trackb

## 二、方法论：核心思想与技术细节

### **总体思路**
提出 **多阶段优化管线**，结合 **YOLOv8 检测模型** 与 **U-Net 分割模型** 的多模型集成策略，通过后处理优化提升准确率与鲁棒性。

### **核心步骤**

1. **YOLO 模型训练与预处理**
   - 原始数据中所有标注均为固定的 100×100 像素框，不符合实际细胞形态。
   - 作者通过重新调整标注框尺寸（10×10 到 120×120），保持中心点不变，以改善定位。
   - 选用 YOLOv8 的 Nano、Small、Medium 三种模型版本进行实验，发现 20×20 和 50×50 尺寸最优。
   - 模型训练参数：输入分辨率 1024×1024，批次大小 16，训练 150 epoch。

2. **模型集成（Ensemble）策略**
   - **第一阶段：YOLO 集成**
     - 集成两个 YOLOv8n 模型（20×20 与 50×50），通过欧氏距离 (≤12 像素) 判断同一细胞重叠预测。
     - 双模型同时预测的结果进行平均坐标和置信度；单模型预测需置信度高于 0.35 才保留。
   - **第二阶段：YOLO + U-Net 集成**
     - U-Net 使用 ResNet34 主干，作为 **热图回归模型**。
     - 标注点被转换为高斯核热图（σ = 期望框大小的 1/6），以 MSE 损失优化。
     - 采用多尺度推理（0.8×、1.0×、1.2×），通过局部最大抑制提取中心点。
     - 最终融合时对齐空间重叠的预测点，平均置信度，置信阈设为 0。确保 YOLO 和 U-Net 的全部检测结果被保留。

3. **三阶段优化管线（后处理）**
   - **阶段1：非极大值抑制 (NMS)**
     - IoU 阈值 0.75，可保留密集细胞邻近的预测，提升 mAP50-95。
   - **阶段2：空间密度过滤**
     - 将图像划分为 4×4 网格。
     - 高密度区域（>30检出）启用更严格置信度 (≥0.1)；
       低密度区域放宽阈值 (≥0.001)，避免漏检病变细胞。
   - **阶段3：二分类器精炼**
     - 使用 EfficientNet-B0 作为二分类过滤器，区分“真实细胞”与“背景伪影”。
     - 通过 Hard Negative Mining 构造训练集：
       - GT 区域标记为正样本；
       - IoU < 0.1 的伪预测作为负样本。
     - 使用加权二元交叉熵训练，推理时以置信度 0.05 为筛选界限。
