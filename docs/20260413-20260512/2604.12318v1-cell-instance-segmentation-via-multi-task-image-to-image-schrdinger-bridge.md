## 摘要
现有的细胞实例分割流程通常将确定性预测与后处理相结合，这种方式对实例掩膜的全局结构施加的显式约束有限。在本研究中，我们提出了一种多任务图像到图像的薛定谔桥（Schrödinger Bridge）框架，将实例分割表述为基于分布的图像到图像生成问题。通过反距离图引入了边界感知的监督机制，并采用确定性推理以获得稳定的预测结果。在 PanNuke 数据集上的实验结果表明，该方法在无需依赖 SAM 预训练或额外后处理的情况下，取得了具有竞争力或更优的性能。此外，在 MoNuSeg 数据集上的补充实验显示了该方法在训练数据有限条件下的鲁棒性。这些结果表明，基于薛定谔桥的图像到图像生成为细胞实例分割提供了一种有效的框架。

## Abstract
Existing cell instance segmentation pipelines typically combine deterministic predictions with post-processing, which imposes limited explicit constraints on the global structure of instance masks. In this work, we propose a multi-task image-to-image Schrödinger Bridge framework that formulates instance segmentation as a distribution-based image-to-image generation problem. Boundary-aware supervision is integrated through a reverse distance map, and deterministic inference is employed to produce stable predictions. Experimental results on the PanNuke dataset demonstrate that the proposed method achieves competitive or superior performance without relying on SAM pre-training or additional post-processing. Additional results on the MoNuSeg dataset show robustness under limited training data. These findings indicate that Schrödinger Bridge-based image-to-image generation provides an effective framework for cell instance segmentation.

## 一、研究核心问题与动机

- **问题背景**：  
  细胞实例分割（Cell Instance Segmentation）是生物医学图像分析的基础任务，但面临以下难点：
  - 显微图像中细胞密集、形态复杂，导致相邻细胞难以分离；
  - 主流方法（如 U-Net, Cellpose, StarDist, CellViT）往往依赖复杂的 **后处理步骤**（如 watershed、形态学运算、聚类），这些步骤具有高度的 **超参数敏感性** 与 **数据集依赖性**；
  - 现有模型大多采用确定性预测，对生成掩膜的全局形态约束不足，容易产生“合并细胞”或“伪碎片”等形态不合理的输出。

- **研究动机**：  
  作者希望从 **分布建模（distribution-based modeling）** 的角度重新表述分割问题，通过生成模型让输出自身具备形态约束，从而**替代或减少手工后处理依赖**，并实现结构一致、形态合理的实例掩膜生成。

## 二、方法论：多任务图像到图像薛定谔桥（Multi-Task Image-to-Image Schrödinger Bridge, MT-I²SB）

### 1. 核心思想

- 将细胞实例分割转化为一个 **基于分布的图像到图像生成问题**；
- 使用 **Schrödinger Bridge (SB)** 理论来连接输入图像分布 \( X_1 \) 与目标分割掩膜分布 \( X_0 \)，形成一种**结构保持（structure-preserving）**的生成过程；
- 在训练中同时学习两个输出：
  - **实例掩膜（instance mask）**
  - **反向距离图（reverse distance map）** —— 提供边界感知监督，增强分割边界识别能力；
- 在推理阶段采用 **确定性推理（deterministic inference）**，以保证输出稳定、无随机噪声波动。

### 2. 模型结构与训练机制

- 模型基于 **U-Net 骨干网络**，以 SB 流程模拟从 *输入图像* 到 *掩膜分布* 的转移过程；
- 定义输入与输出状态：
  - \( X_1 = (X_i, X_i) \)（图像复制 2 次，共 6 通道）
  - \( X_0 = (M_i, M_i, M_i, R_i, R_i, R_i) \)（掩膜与反距离图，各复制 3 次，共 6 通道）
- 中间状态 \( X_t \) 遵循解析后验：
  \[
  q(X_t | X_0, X_1) = \mathcal{N}(X_t; \mu_t, \Sigma_t)
  \]
  其中 \(\mu_t, \Sigma_t\) 由噪声调度参数 \(\sigma_t, \bar{\sigma}_t\) 控制。
- **损失函数**：
  \[
  \mathcal{L} = \|\epsilon_\theta(X_t, t) - (X_t - X_0) / \sigma_t\|_2^2
  \]
  用于驱动样本点沿 SB 轨迹向目标图像迁移。
- **推理阶段**：禁用噪声注入，沿最可能轨迹反向生成；最终输出三通道掩膜平均并通过阈值 0.5 得到预测。

### 3. 反向距离图机制

- 用 **欧氏距离变换（EDT）** 计算每个像素到边界的距离；
- 取其归一化反值，使 **靠近边界的像素值较高**；
- 与二值掩膜联合训练，以强化网络对边界形态的建模能力；
- 推理阶段不使用反距离图，仅辅助训练。

## 三、实验设计

### 1. 数据集

- **PanNuke**：  
  - 大规模组织学细胞分割数据集，7,904 张图像，19 种组织类型，189,744 个细胞；用于**大规模训练评估**。
- **MoNuSeg**：  
  - 多器官核分割数据集，30 张训练图像 + 14 张测试图像，约 22,000 个细胞；代表**小样本/数据稀缺**场景。

| 对比模型 | 说明 |
|-----------|------|
| **U-Net** | 标准确定性分割基线 |
| **U-Net+R** | 加入反距离图多任务监督 |
| **CellViT** | 基于 Vision Transformer 的最新模型，使用 SAM 预训练 |
| **CellViT+proc** | 同上，但加入官方后处理流水线 |
| **Ours** | 本文所提分布式生成框架 |

### 3. 实验指标  
- **bPQ（binary Panoptic Quality）** – 综合检测与分割精度；  
- **F1-score、Recall、Precision** – 通过重心匹配（12 像素半径）计算识别性能。
