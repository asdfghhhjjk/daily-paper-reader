## 摘要
空间转录组学能够描绘空间基因表达谱，但受限于高成本和低通量，这促使人们从H&E组织病理学图像中预测基因表达。现有的上下文感知方法主要监督绝对表达，而很少显式利用点之间的相对表达关系。我们提出了COAST，一个面向空间基因表达预测的上下文感知差分学习框架。COAST通过类型特定调制对局部和全局上下文特征进行条件化，并使用Transformer编码器聚合目标和上下文点的标记，以捕捉细粒度的局部模式和切片级结构。其训练采用联合目标，将绝对表达回归与目标和上下文点之间的符号差分回归相结合。在多个空间转录组学数据集上的实验显示，基于相关性和分布的评估指标均有持续改进，证明了上下文感知差分学习在基于组织学的空间基因表达预测中的有效性。

## Abstract
Spatial transcriptomics enables profiling of spatial gene expression but is limited by high cost and low throughput, motivating prediction from H&E histopathology images. Existing context-aware methods mainly supervise absolute expression, while relative expression relationships between spots are rarely used explicitly. We propose COAST, a context-aware differential learning framework for spatial gene expression prediction. COAST conditions the local and global context features with type-specific modulation and aggregates the target and context spot tokens using a Transformer encoder to capture both fine-grained local patterns and slide-level structure. It is trained with a joint objective that combines absolute expression regression with signed differential regression between the target and context spots. Experiments on multiple spatial transcriptomics datasets show consistent improvements in correlation- and distribution-based metrics, demonstrating the effectiveness of context-aware differential learning for histology-based spatial gene expression prediction.

- **整体含义**：作者提出COAST框架，旨在显式建模目标点与上下文点之间的有符号表达差异，通过联合绝对回归与差分回归，增强模型对空间关系的捕捉能力，提升预测的鲁棒性和生物学可解释性。

### 2. 论文提出的方法论：核心思想、关键技术细节
- **整体框架**：COAST由四部分组成：特征提取器、上下文特定特征调制、空域关系Transformer、绝对与差分预测头。  
- **特征提取与上下文采样**：对每个点使用预训练特征提取器（UNI）得到特征向量x_t。针对目标点t，从全切片图像中采样K个上下文点，分为局部上下文（由k最近邻得到，数量K_L）和全局上下文（随机采样或网格采样，数量K_G）。每个上下文点带有类型标签c_k（0为局部，1为全局）。  
- **上下文特定调制**：将目标和上下文特征通过共享的编码器头Φ_head映射为隐空间向量h。为解决上下文特征丧失空间类型信息的问题，引入两组可学习的缩放参数γ和偏移参数β（分别对应局部和全局类型），通过嵌入层生成，对上下文特征进行仿射调制：  
  \[
  \tilde{h}_k = h_k \odot (1 + \gamma_{c_k}) + \beta_{c_k}
  \]  
  该操作使模型能根据上下文类型自适应调节特征。  
- **空域关系Transformer**：将目标token与所有调制后的上下文token拼接，输入标准L层Transformer编码器，输出上下文感知的表示{z_t, z_1, ..., z_K}。  
- **预测头与联合损失**：  
  - 经共享投影头Φ_proj得到最终表征f_t和上下文表征f_k。  
  - 回归头ϕ_reg预测绝对表达ŷ_t = ϕ_reg(f_t)。  
  - 差分头ϕ_diff从特征差(f_t - f_k)预测表达差异Δŷ_{t,k} = ϕ_diff(f_t - f_k)。  
  - 损失函数：  
    \[
    \mathcal{L} = \|y_t - \hat{y}_t\|^2 + w_{\text{diff}} \cdot \frac{1}{K} \sum_{k=1}^K \|\Delta y_{t,k} - \Delta \hat{y}_{t,k}\|^2
    \]  
    其中Δy_{t,k} = y_t - y_k为真实差异，w_diff为权重。  
- **核心思想**：通过同时回归绝对表达和带方向的相对差异，模型学习到更稳定的空间梯度，缓解批次效应。

### 3. 实验设计：数据集、Benchmark、对比方法
- **数据集**：使用SpaRED提供的7个公共空间转录组数据集，均含128个目标基因和预定义测试划分：Abalo人鳞状细胞癌(AHSCC)、Erickson人前列腺癌P1(EHPCP1)、Mirzazadeh小鼠骨(MMBO)、Mirzazadeh小鼠脑P1/P2(MMBP1, MMBP2)、Vicari小鼠脑(VMB)、Villacampa肺类器官(VLO)。数据为224×224 H&E斑块，表达值经TPM标准化和log1p变换。  
- **评估指标**：基因级平均的皮尔逊相关系数(PCC)、互信息(MI)、区分零与非零表达的ROC-AUC(AUC 0vNZ)、区分中位数上下表达的ROC-AUC(AUC Q50)、归一化均方根误差(NRMSE)。  
- **对比方法**：6种代表性基线模型：STNet、Hist2ST、TCGN、EGNv2、NH2ST、PEKA，均采用官方实现及默认超参数。  
- **下游任务**：在TCGA-LUAD队列上预测总体生存期(OS)，比较仅使用图像特征、图像+PEKA预测表达、图像+COAST预测表达三种配置，以C-index衡量，并与bulk RNA测序的Cox模型对照。
