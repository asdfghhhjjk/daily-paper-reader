## 摘要
弱监督全切片图像分类在计算病理学中应用广泛，因为切片级标签比密集区域标注更容易获得。现有的多实例学习方法通常主要利用视觉线索聚合大量图像块嵌入，这可能保留许多非信息性图像块，并在实例特征与类别级疾病语义之间提供弱对齐。我们提出概念引导剪枝与表示学习（CGRL），一个简单框架，将源自疾病提示的类别级概念原型引入多实例学习流程。首先，概念相关性剪枝根据图像块与类别概念的相似度对图像块实例进行排序，并保留前K个概念相关图像块用于下游多实例聚合。其次，概念引导对比表示学习从同一相似度矩阵构建类别正负图像块集，并优化目标类别、对称辅助和跨类分离目标，从而规范化投影概念空间。我们在TCGA-BRCA和TCGA-NSCLC数据集上使用多种代表性多实例学习方法评估CGRL。实验结果表明，CGRL在多个模型-数据集组合上有性能提升，增益取决于下游多实例模型和数据集。它在准确率和宏F1值上取得了特别明显的改进，同时通过概念相关性剪枝降低了计算成本。这些发现表明，类别级语义概念为弱监督计算病理学中的图像块选择和表示学习提供了有效且实用的先验。

## Abstract
Weakly supervised whole-slide image (WSI) classification is widely used in computational pathology because slide-level labels are easier to obtain than dense region annotations. Existing multiple instance learning (MIL) methods often aggregate large bags of patch embeddings using mainly visual cues, which can retain many non-informative patches and provide weak alignment between instance features and class-level disease semantics. We propose Concept-Guided Pruning and Representation Learning (CGRL), a simple framework that introduces class-level concept prototypes derived from disease prompts into the MIL pipeline. First, concept-relevance pruning ranks patch instances by their similarity to class concepts and retains the top-K concept-relevant patches for downstream MIL aggregation. Second, concept-guided contrastive representation learning constructs class-wise positive and negative patch sets from the same similarity matrix and optimizes target-class, symmetric auxiliary, and cross-class separation objectives, thereby regularizing the projected concept space. We evaluate CGRL on TCGA-BRCA and TCGA-NSCLC using multiple representative MIL methods. Experimental results show that CGRL improves several model-dataset combinations, with gains depending on the downstream MIL model and dataset. It achieves particularly clear improvements in accuracy and macro-F1 while reducing computational cost through concept-relevance pruning. These findings demonstrate that class-level semantic concepts provide an effective and practical prior for patch selection and representation learning in weakly supervised computational pathology.

- **研究背景**：计算病理学中，全切片图像分类对癌症诊断与分型至关重要，但获取密集区域标注成本高昂，因此弱监督多实例学习成为主流范式。

  - **语义对齐弱**：实例特征与高层疾病语义（如亚型间的细微形态差异）之间缺乏显式对齐。
- **整体含义**：提出利用文本疾病提示构建类别级语义概念原型，通过概念相关性剪枝与概念引导对比学习，将语义先验注入弱监督 MIL 流程，实现更精确、更高效的切片级分类。

## 2. 论文提出的方法论
- **核心思想**：引入冻结的视觉-语言模型提取的类别概念原型，以统一的方式为 MIL 提供**斑块选择**与**表示学习**的双重语义指导。
- **关键技术细节**：
  1. **斑块与概念嵌入**：
     - 使用 CONCH 图像编码器提取斑块特征 $x_i$。
     - 使用 TITAN 文本编码器从疾病提示（遵循 CATE 的提示构造策略）得到类别概念原型 $t_c$。
     - 通过可学习投影头 $g_\phi$ 将斑块特征映射到概念空间，得到投影特征 $u_i$。
  2. **概念相关性剪枝**：
     - 计算投影特征与原型之间的余弦相似度矩阵 $S$，元素 $S_{i,c}$。
     - 定义每个斑块的概念相关性分数 $r_i = \max_c S_{i,c}$。
     - 保留得分最高的 $K_p$ 个斑块（本文 $K_p=300$），构成剪枝后的包，输入下游 MIL 模型。
  3. **概念引导对比表示学习**（仅训练时使用）：
     - 基于 $S$ 为每个类别 $c$ 选择 $K_{pos}$ 个最相关和 $K_{neg}$ 个最不相关的投影特征索引。
     - 构建三个 InfoNCE 对比损失：
       - **目标类损失 $\mathcal{L}_y$**：使真实标签类的相关投影特征与原型 $t_y$ 对齐，远离该类不相关特征。
       - **对称辅助损失 $\mathcal{L}_{sym}$**：为非目标类 $c \ne y$ 执行相似的对齐与分离，正则化整个概念空间。
       - **跨类分离损失 $\mathcal{L}_{cross}$**：增强类间鉴别性，使目标类相关投影特征接近“非目标类不相关特征”的均值表示 $p_{y \to c}$，同时远离目标类不相关特征和竞争类相关特征。
     - 总损失 $\mathcal{L}_{CGRL} = \mathcal{L}_y + \lambda_{sym}\mathcal{L}_{sym} + \lambda_{cross}\mathcal{L}_{cross}$，与 MIL 分类损失联合优化。
- **推理流程**：仅保留概念相关性剪枝步骤，移除对比学习组件，因此 CGRL 是一种轻量级的即插即用模块。

## 6. 论文的主要结论与发现
- **斑块采样的普遍益处**：在全切片分类中，通过采样移除冗余斑块通常能提升性能，验证了去冗余的动机。
- **CGRL 在阈值依赖指标上的优势**：概念引导剪枝在准确率和宏 F1 上取得了显著且一致的提升，尤其在注意力型 MIL 模型上表现突出（例如 ABMIL+ CGRL 在 BRCA 上的 ACC 从 0.9101 提升至 0.9307）。
- **效果依赖模型与数据集**：对某些 MIL 模型和数据集，概念引导采样增益更明显，而随机或 K-means 采样在某些指标上仍有竞争力，表明语义先验与聚合策略的交互影响最终表现。
- **语义先验增强类间鉴别**：CGRL 在强调类别区分性指标上的优势表明，类别概念原型能有效正则化特征空间，提升亚型间细微差异的辨别能力。
- **计算效率提升**：通过剪枝大幅减少输入斑块数量，在保持或提升性能的同时降低了 MIL 聚合器的计算开销。

## 7. 优点
- **轻量级模块化设计**：即插即用，无需改动下游 MIL 架构，推理时仅增加剪枝步骤，无额外推理时间开销。
- **双重语义利用**：同时将概念原型用于斑块选择和对比监督，设计统一且互补。
- **缓解两大痛点**：直接针对冗余和语义对齐弱的问题，提供了简洁有效的解决方案。
- **评估全面**：在多个主流 MIL 方法、两个数据集上验证，并包含消融和计算量分析，说服力较强。
- **可解释性潜力**：概念相关性分数可为哪些斑块对诊断重要提供直观解释。

## 8. 不足与局限
- **模型依赖性**：性能提升并非在所有 MIL 模型上一致，部分场景下增益有限，通用性仍需增强。
