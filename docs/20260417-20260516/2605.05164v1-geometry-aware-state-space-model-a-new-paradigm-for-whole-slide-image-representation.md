## 摘要
对组织病理图像的精确分析对于疾病诊断和治疗方案制定至关重要。全切片图像（Whole-Slide Images, WSIs）以千兆像素分辨率数字化组织切片，是这一过程的基础，但在进行切片级预测时需要聚合成千上万个图像块（patch）。多实例学习（Multiple Instance Learning, MIL）通过采用两阶段范式来应对这一挑战，将图块级特征嵌入与切片级预测解耦。然而，大多数现有方法隐式地将图块表示嵌入同质的欧几里得空间中，忽视了病理组织的层次化结构和区域异质性。这限制了当前模型对全局组织结构与细粒度细胞形态的捕捉能力。为克服这一局限，我们提出了一种混合双几何表示方法，将WSI特征嵌入双重几何空间（双曲-欧几里得空间），从而能够互补地建模组织的层级结构和局部形态细节。基于此方法，我们开发了WSI分类框架BatMIL，该框架同时利用这两种几何空间。为了建模数千个图块之间的长程依赖关系，我们采用结构化状态空间序列模型（Structured State Space Sequence Model, S4）作为主干网络，以线性计算复杂度对图块序列进行编码。此外，为了处理区域异质性，我们引入了一个基于块级的专家混合（Mixture-of-Experts, MoE）模块，将图块按区域分组并动态分配至专用子网络，从而在提升表示能力的同时减少冗余计算。在涵盖六种癌症类型的七个WSI数据集上的大量实验表明，BatMIL在切片级分类任务中持续超越当前最先进的MIL方法。结果显示，具备几何感知能力的表示学习为下一代计算病理学提供了有前景的方向。

## Abstract
Accurate analysis of histopathological images is critical for disease diagnosis and treatment planning. Whole-slide images (WSIs), which digitize tissue specimens at gigapixel resolution, are fundamental to this process but require aggregating thousands of patches for slide-level predictions. Multiple Instance Learning (MIL) tackles this challenge with a two-stage paradigm, decoupling tile-level embedding and slide-level prediction. However, most existing methods implicitly embed patch representations in homogeneous Euclidean spaces, overlooking the hierarchical organization and regional heterogeneity of pathological tissues. This limits current models' ability to capture global tissue architecture and fine-grained cellular morphology. To address this limitation, we introduce a hybrid hyperbolic-Euclidean representation that embeds WSI features in dual geometric spaces, enabling complementary modeling of hierarchical tissue structures and local morphological details. Building on this formulation, we develop BatMIL, a WSI classification framework that leverages both geometric spaces. To model long-range dependencies among thousands of patches, we employ a structured state space sequence model (S4) backbone that encodes patch sequences with linear computational complexity. Furthermore, to account for regional heterogeneity, we introduce a chunk-level mixture-of-experts (MoE) module that groups patches into regions and dynamically routes them to specialized subnetworks, improving representational capacity while reducing redundant computation. Extensive experiments on seven WSI datasets spanning six cancer types demonstrate that BatMIL consistently outperforms state-of-the-art MIL approaches in slide-level classification tasks. These results indicate that geometry-aware representation learning offers a promising direction for next-generation computational pathology.

## 一、核心问题与研究背景

- **研究领域**：计算病理学中的全切片图像（Whole-Slide Image, WSI）分析。  
- **问题本质**：WSI 具有千兆像素级分辨率，一个切片可能包含数千个局部图块（patch），需要在全局层次上综合判断病灶特征。  
- **现有解决方案的局限**：
  - 多实例学习（MIL）方法已成为主流解决方案，通过分离图块级嵌入与切片级聚合实现可扩展建模。
  - 但大多数MIL方法仅在**欧几里得空间**中嵌入特征，无法反映病理组织的**层次结构（细胞—腺体—组织）**与**区域异质性（肿瘤区 vs 正常区）**。
  - 结果导致模型倾向过度聚焦局部特征，难以捕捉全局组织架构或区分病变区域。

**研究目标**：突破欧几里得空间的约束，提出一种能同时捕捉组织层级与局部形态细节的新型几何表征方法，提升全切片图像分类性能和解释性。

## 二、方法论：BatMIL 框架

### 2.1 总体结构
论文提出 **BatMIL** 框架，全称 *Geometry-Aware State Space Model-based Multiple Instance Learning*。

BatMIL 的核心思想：
- 在**双几何空间（双曲空间 + 欧几里得空间）**中嵌入特征，实现对层次和局部信息的互补建模；
- 结合**S4 状态空间模型**与**混合专家（MoE）模块**，高效建模超长序列（上千图块）及解耦组织异质性。

### 2.2 模型主要组成

#### （1）WSI 预处理与 patch 生成
- 分辨率：0.5 microns-per-pixel。
- 步骤：亮度阈值(Otsu) → 去除低质量或空白图块 → 使用基础模型（如 GigaPath）嵌入得到语义特征向量。

#### （2）Structured State Space Sequence（S4）模块
- 功能：以**线性复杂度**建模数千图块间的长程依赖。
- 技术机制：
  - 采用 HiPPO 初始化以保持长期记忆稳定；
  - 引入 DropPath 残差连接，多层归一化；
  - 能在无序二维布局中建模全局上下文，不依赖顺序遍历（相较 Transformer 更符病理图像空间特征）。

数学表达（简述）：
> x_S4 = x + DropPath(S4(LN(x)))

#### （3）Mixture-of-Experts（MoE）模块
- 功能：处理WSI内的区域异质性。
- 实现：
  - 包含 **4 专家网络**（两层前馈，含 GELU 激活与 dropout）。
  - 模型通过 gating 网络为每个图块选择前 2 名专家（top-2 routing）。
  - 动态分配权重并融合输出。
- 效果：赋予不同区域专用的表征能力，避免参数稀释与背景干扰。

#### （4）Hybrid Geometric Representation（双几何表示）
- 欧几里得空间：保留微观形态细节；
- 双曲空间：刻画层次型结构（利用 Poincaré 球模型，曲率 c 控制嵌入特性）。
- 关键融合策略：**加权相加（weighted addition）**，由可学习参数 α 逐元素平衡两种几何空间。
  
公式（文字化描述）：
> z_hybrid = α · z_Euclidean + (1 - α) · log_map(z_Hyperbolic)

该融合同时保持低维可解释性和结构一致性，是本研究提出的核心创新之一。

## 六、主要结论与发现

- BatMIL在七个数据集上的平均性能均超过十三种SOTA方法。
- 双几何空间嵌入有效改善病理组织的层次表达与局部准确性。
- 加权加法融合方式在GHS模块中表现最佳，比简单拼接或映射投影具更稳定性能。
- MoE机制能动态分配专家，提升异质区域识别效果并降低计算冗余。
- 最优曲率值为 **c = 0.1**，在层级建模与局部细节保真之间实现平衡。
- 模型在效率与性能间取得良好折衷：参数量仅9.76M，但性能显著高于大型Transformer模型（如86M参数的GigaPath）。
