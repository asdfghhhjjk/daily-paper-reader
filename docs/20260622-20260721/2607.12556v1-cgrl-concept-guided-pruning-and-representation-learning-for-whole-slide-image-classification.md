## 摘要
弱监督全切片图像（WSI）分类在计算病理学中广泛应用，因为切片级标签比密集区域标注更容易获取。现有的多实例学习（MIL）方法通常主要依赖视觉线索聚合大量补丁嵌入，这可能保留许多非信息性补丁，并导致实例特征与类别级疾病语义之间的弱对齐。我们提出概念引导的剪枝与表示学习（CGRL），一个简单的框架，将从疾病提示中派生的类别级概念原型引入MIL流程。首先，概念相关度剪枝根据补丁实例与类别概念的相似度对补丁排序，并保留前K个概念相关补丁用于下游MIL聚合。其次，概念引导的对比表示学习从同一相似度矩阵构建逐类的正负补丁集，并优化目标类别、对称辅助和跨类别分离目标，从而规范化投影概念空间。我们在TCGA-BRCA和TCGA-NSCLC数据集上，使用多种代表性MIL方法评估了CGRL。实验结果显示，CGRL改善了多个模型-数据集组合，增益取决于下游MIL模型和数据集。它在准确率和宏F1分数上取得了特别明显的提升，同时通过概念相关度剪枝降低了计算成本。这些发现表明，类别级语义概念为弱监督计算病理学中的补丁选择和表示学习提供了有效且实用的先验。

## Abstract
Weakly supervised whole-slide image (WSI) classification is widely used in computational pathology because slide-level labels are easier to obtain than dense region annotations. Existing multiple instance learning (MIL) methods often aggregate large bags of patch embeddings using mainly visual cues, which can retain many non-informative patches and provide weak alignment between instance features and class-level disease semantics. We propose Concept-Guided Pruning and Representation Learning (CGRL), a simple framework that introduces class-level concept prototypes derived from disease prompts into the MIL pipeline. First, concept-relevance pruning ranks patch instances by their similarity to class concepts and retains the top-K concept-relevant patches for downstream MIL aggregation. Second, concept-guided contrastive representation learning constructs class-wise positive and negative patch sets from the same similarity matrix and optimizes target-class, symmetric auxiliary, and cross-class separation objectives, thereby regularizing the projected concept space. We evaluate CGRL on TCGA-BRCA and TCGA-NSCLC using multiple representative MIL methods. Experimental results show that CGRL improves several model-dataset combinations, with gains depending on the downstream MIL model and dataset. It achieves particularly clear improvements in accuracy and macro-F1 while reducing computational cost through concept-relevance pruning. These findings demonstrate that class-level semantic concepts provide an effective and practical prior for patch selection and representation learning in weakly supervised computational pathology.

## 1. 论文的核心问题与整体含义
论文针对弱监督全切片图像（WSI）分类中的两个固有问题：
- **补丁冗余**：一张WSI切出数万图像补丁，大量非信息性补丁增加计算开销，并可能稀释关键诊断信号。
- **语义对齐弱**：传统MIL方法仅依赖视觉特征聚合，实例特征与高层疾病概念（如“浸润性导管癌”）缺乏显式关联，尤其当亚型依赖细微形态学差异时表现不足。

提出的CGRL框架利用从疾病文本提示中编码的类别级概念原型，同时指导**补丁剪枝**和**表示学习**，旨在以可插拔的方式提升现有MIL方法的准确率并降低计算成本。

## 2. 方法论
### 2.1 核心思想
以冻结的视觉和文本编码器为基础，为每个类别建立一个概念原型，通过相似度矩阵同时实现：
- **概念相关度剪枝**：只保留与任意类别概念最相似的前K个补丁，过滤冗余信息。
- **概念引导对比学习**：用相似度排序构造正负样本对，设计三类对比损失，强化补丁表示的概念语义结构。

### 2.2 关键技术细节
- **概念原型构建**：使用预训练的TITAN文本编码器，对每个类别（如“浸润性导管癌”、“浸润性小叶癌”）的提示文本编码，获得类别原型 \(T = \{t_c\}_{c=1}^C\)。
- **补丁投影与相似度计算**：用一个可学习的投影头 \(g_\phi\) 将补丁嵌入 \(x_i\) 映射到概念空间 \(u_i\)，计算 \(\ell_2\) 归一化后的余弦相似度矩阵 \(S\)，其中 \(S_{i,c} = \cos(u_i, t_c)\)。
- **剪枝策略**：每个补丁取跨类最大相似度 \(r_i = \max_c S_{i,c}\) 作为概念相关度分数，保留分数最高的 \(K_p\) 个补丁，构成压缩包 \(\tilde{X}\) 送入下游MIL模型。补丁顺序保持原序。
- **对比学习样本构造**：对每个类别 \(c\)，在原始相似度列 \(S_{:,c}\) 上取前 \(K_{pos}\) 个补丁作为正样本索引 \(P_c\)，取后 \(K_{neg}\) 个作为负样本索引 \(N_c\)。
- **三类对比损失**（采用InfoNCE）：
  - **目标类损失 \(\mathcal{L}_y\)**：鼓励真实标签 \(y\) 对应的正样本与原型 \(t_y\) 对齐，远离负样本。
  - **对称辅助损失 \(\mathcal{L}_{sym}\)**：对每个非目标类 \(c\neq y\)，同样让该类的正样本向 \(t_c\) 靠拢，远离负样本，提升整体概念空间的可分性。
  - **跨类分离损失 \(\mathcal{L}_{cross}\)**：对每个非目标类 \(c\)，用负样本的均值构建辅助原型 \(p_{y\to c}\)，促使目标类正样本接近该辅助原型，同时远离非目标类正样本，增强跨类判别力。
- **总损失**：\(\mathcal{L} = \mathcal{L}_{cls} + \lambda_{CGRL} (\mathcal{L}_y + \lambda_{sym}\mathcal{L}_{sym} + \lambda_{cross}\mathcal{L}_{cross})\)。
- **推理阶段**：移除所有对比损失，仅保留概念相关度剪枝 + MIL分类，轻量化部署。

## 6. 论文的主要结论与发现
- 在**BRCA**上，CGRL与ABMIL和CLAM-SB组合时，ACC/F1提升最明显（例如ABMIL：ACC从0.9101到0.9307，F1从0.8449到0.8781；CLAM-SB：ACC/F1分别达到0.9400/0.8982）。

- 结论：类别级语义概念作为先验，能以即插即用的方式，同时提升下游模型的判别力和计算效率，但提升幅度与MIL聚合策略相关。

## 7. 优点
- **双重概念利用**：同一个相似度矩阵既用于补丁筛选又用于对比正则化，设计紧凑。
- **即插即用**：冻结视觉和文本编码器，只需训练投影头和MIL部分，易于集成到现有MIL方法。
- **推理高效**：对比损失仅用于训练，推理时仅做一次相似度计算和Top-K选择，不增加深度网络计算。
- **语义可解释性强**：剪枝直接基于补丁与疾病概念的相似度，有助于暴露模型关注的病理语义区域。
- **实验基准较全**：覆盖多种经典MIL方法，并与随机、Kmeans等采样方法对比，验证了概念引导的优势并非单纯来自降采样。
