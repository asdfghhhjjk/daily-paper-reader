## 摘要
从常规H&E染色组织学图像中表征肿瘤微环境（TME）需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出了SegTME-UNI2，一个满足这些需求的统一框架。其核心是UNI2-UperHoVeR，一个双头分割模型，将UNI2-h病理学基础模型（ViT-Giant，在10万张切片、超过1亿个图块上预训练）与两个并行的UperNet解码器相结合：一个用于六类语义分割，另一个用于水平-垂直梯度回归以实现基于分水岭的细胞核实例分割。为了解决大型真实世界数据集中缺乏像素级标注的问题，UNI2-UperHoVeR经历了一个三阶段渐进式伪标签课程学习。每个阶段都训练一个全新的模型而不进行权重迁移，完全通过提高伪标签质量来推动改进：第一阶段：使用人工标注的PanNuke数据集（7,901张图像，189,744个细胞核，0.25微米/像素）。第二阶段：使用第一阶段模型在271,711个TCGA-UT比例尺0的图块（0.5微米/像素）上生成的、经过熵过滤的伪标签。第三阶段：使用第二阶段模型在全部1,608,060个TCGA-UT图块（跨越六个分辨率尺度，0.5-1.0微米/像素）上生成的伪标签。分割结果输入到一个结构化的TME特征提取流程，该流程计算每个图块的20多个组成、形态、空间熵和细胞间距离指标。这些指标被编码为JSON并传递给一个微调过的NVIDIA BioNeMo GPT模型，以生成临床可解释的TME描述。在留出的PanNuke和TCGA-UT划分上的初步验证证明了框架的可行性和内部一致性。伪标签TCGA-UT数据集和UNI2-UperHoVeR模型检查点已公开发布，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SegTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UperHoVeR, a dual-head segmentation model pairing the UNI2-h pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UperHoVeR undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UperHoVeR checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

### 2. 方法论
#### 2.1 核心架构：UNI2‑UperHoVer
- **共享骨干网络**：病理基础模型 UNI2‑h（ViT‑Giant，在 >1 亿图块上预训练），从第 5、11、17、23 个 Transformer 块提取四尺度特征金字塔（步长 4‑32）。
- **双头 UperNet 解码器**：
  - **六类语义分割头**（Neoplastic， Inflammatory， Connective， Dead， Epithelial， Background），输出类别概率。
  - **HV 回归头**（2 通道），预测核内像素的质心方向距离梯度图，作为分水岭实例分割的能量面。
- **动态 HV 目标合成**：仅需语义掩码，通过连通域分析计算每个细胞核质心，生成归一化到 [-1, 1] 的水平和垂直方向图，从而在无实例级人工标注的情况下训练实例分离能力。
- **损失函数**：语义交叉熵损失 + HV 回归损失（MSE + 梯度 MSE，权重 2:1），总损失直接相加（λ = 1.0）。

#### 2.2 三阶段渐进式伪标签课程学习
- **阶段一（种子训练）**：在人工标注的 PanNuke（7,901 张图像，0.25 μm/px）上训练模型 θ1。
- **阶段二（域适应）**：θ1 在 271,711 张 TCGA‑UT 尺度 0（0.5 μm/px）图块上生成伪标签，经熵阈值过滤后，从零初始化训练模型 θ2（不继承 θ1 权重）。
- **阶段三（多尺度扩展）**：θ2 在全部 1,608,061 张 TCGA‑UT 六个尺度（0.5‑1.0 μm/px）上生成伪标签，训练最终模型 θ3。
- **关键设计**：每个阶段均从头训练，伪标签质量的逐次提升直接反映在下一阶段更高的初始 mIoU 上（0.41→0.47→0.53），证明课程学习的有效性。

#### 2.3 TME 特征提取与 LLM 叙述生成
- **特征提取流水线**：基于语义和实例掩码，计算每图块 20+ 定量指标（细胞计数/占比、Tumor‑Stroma 比、免疫浸润比、基于 k‑d 树的免疫‑肿瘤距离、局部空间熵等），输出结构化 JSON。
- **生物 token 转换**：根据阈值将数值特征映射为类别标签和 token 列表（如“Immune Hot”、“Desmoplastic”等）。
- **Narrative 生成**：使用 NVIDIA BioNeMo 对 LLaMA‑3.2‑1B 进行监督微调（SFT），输入为结构化提示（含 METADATA、RAW_FEATURES、SCENARIO、KEYWORDS），自回归生成临床通顺的 TME 段描述，且每项声明均可追溯至数值特征，降低幻觉风险。

### 6. 主要结论与发现
- **三阶段伪标签课程**可在不增加人工标注的条件下，将 PanNuke 学到的核分割能力扩展到 1.6M 张 TCGA 实图，且每个阶段新模型均获得持续改善的初始 mIoU。

### 7. 优点
- **统一框架**：首次将大型病理基础模型、多类语义+实例分割、伪标签扩展、空间 TME 指标与 LLM 叙事先连成完整的自动化分析流程。
- **标注高效**：仅需少量语义级标注（PanNuke），通过动态 HV 合成和熵过滤伪标签课程，实现对海量未标注真实数据的分割。
- **鲁棒性设计**：叙事生成依赖于整体统计特征而非单个细胞边界精度，使输出在染色、扫描仪差异下仍保持稳定。

### 8. 不足与局限
- **缺少外部基准评估**：未与 HoVer‑Net、CellPose、StarDist 等主流方法在公开独立测试集上比较，无法衡量分割精度的相对水平。

- **应用局限**：当前仅分析 256×256 图块，未涉及全切片级别上下文；且仅使用 H&E 染色，未融合多模态数据。
