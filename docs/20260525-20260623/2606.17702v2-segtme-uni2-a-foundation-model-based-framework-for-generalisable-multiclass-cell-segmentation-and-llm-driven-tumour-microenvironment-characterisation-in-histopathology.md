## 摘要
从常规H&E染色组织学图像中表征肿瘤微环境（TME）需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出了SegTME-UNI2，一个统一框架以满足这些需求。其核心是UNI2-UperHoVeR，一种双头分割模型，将UNI2-h病理学基础模型（ViT-Giant，在来自10万张切片的1亿多张图块上预训练）与两个并行的UperNet解码器配对：一个用于六类语义分割，另一个用于水平-垂直梯度回归，实现基于分水岭的细胞核实例分离。为解决大型真实世界数据集中像素级标注的缺失，UNI2-UperHoVeR采用了三阶段渐进式伪标签课程。每个阶段训练一个新模型而不进行权重迁移，完全通过提高伪标签质量来推动改进：阶段1：使用人工标注的PanNuke（7901张图像，189744个细胞核，0.25微米/像素）。阶段2：使用阶段1模型在271711个TCGA-UT比例尺0图块（0.5微米/像素）上的熵过滤伪标签。阶段3：使用阶段2模型在全部1608060个TCGA-UT图块（六个分辨率尺度，0.5-1.0微米/像素）上的伪标签。分割输出馈入一个结构化的TME特征提取流水线，计算20多个每图块的组成、形态、空间熵和细胞间距离指标。这些指标被编码为JSON，并传递给微调的NVIDIA BioNeMo GPT模型，以生成临床可解释的TME叙述。在保留的PanNuke和TCGA-UT分区上的初步验证显示了框架的可行性和内部一致性。伪标注的TCGA-UT数据集和UNI2-UperHoVeR检查点已公开发布，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SegTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UperHoVeR, a dual-head segmentation model pairing the UNI2-h pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UperHoVeR undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UperHoVeR checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：从常规H&E染色病理图像中表征肿瘤微环境（TME）需要同时完成三项关键任务：多类细胞分割、生物特征提取和可解释的临床报告生成。现有方法要么仅做实例分割（如HoVer-Net），要么仅做语义分割（如UperNet），无法同时兼顾；且缺乏大规模像素级标注数据，导致模型泛化能力受限。
- **整体含义**：提出SegTME-UNI2框架，通过结合病理基础模型（UNI2-h）、双头分割解码器、渐进式伪标签课程、结构化TME量化特征提取和大语言模型（LLM）叙述生成，实现从原始图像到临床报告的端到端自动化，旨在弥合人工标注数据与大规模真实世界数据（TCGA）之间的200倍标注差距。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：采用三阶段渐进式伪标签课程，从人工标注的PanNuke数据出发，逐步扩展到超160万张TCGA-UT图块，每阶段训练全新的模型（不继承前阶段权重），依靠伪标签质量提升驱动性能进步；同时设计双头解码器UNI2-UperHoVer，联合优化语义分割和水平-垂直（HV）梯度回归，实现细胞核实例分离。
- **关键技术细节**：
  - **UNI2-UperHoVer架构**：以UNI2-h（ViT-Giant，预训练于>1亿病理图块）为共享骨干，从第5、11、17、23个Transformer块提取四尺度FPN特征；两个独立UperNet解码头分别输出六类语义logits（背景、肿瘤、炎症、结缔组织、死亡、上皮）和两通道HV梯度图。
  - **动态HV目标合成**：从语义掩码通过连通组件分析生成近似HV梯度目标（归一化到[-1,1]），无需实例级标注即可训练。
  - **损失函数**：语义损失采用交叉熵（忽略索引255）；HV损失结合像素级MSE和梯度匹配项（空间梯度误差），总损失为二者之和（权重λ=1.0）。
  - **三阶段课程**：
    - 阶段1：在PanNuke（7901张图，0.25μm/pixel）上进行有监督训练，得到模型M1。
    - 阶段2：M1在TCGA-UT Scale 0（271,711张图，0.5μm/pixel）上生成伪标签，经熵过滤后用于训练M2（从UNI2-h重新初始化）。
    - 阶段3：M2在所有六个尺度（1,608,061张图，0.5-1.0μm/pixel）上生成伪标签，经熵过滤后用于训练最终模型M3。
  - **TME特征提取**：从语义和实例分割结果计算20+指标，包括每类细胞计数、占比、平均核面积、肿瘤-间质分数、免疫浸润分数、空间熵（基于5最近邻的Shannon熵）、免疫-肿瘤细胞最小/平均距离（基于KD树）等。
  - **BioNeMo GPT叙述生成**：将上述特征编码为结构化JSON，通过NVIDIA BioNeMo框架对LLaMA-3.2-1B进行监督微调，输入格式为“METADATA | RAW_FEATURES | SCENARIO | KEYWORDS”，输出临床风格的TME叙述。

## 3. 实验设计：数据集、benchmark、对比方法

  - **TCGA-UT Restructured**：来自32种实体癌类型，8,736张切片，共1,608,061张256×256图块，覆盖6个分辨率尺度（0.5-1.0μm/pixel）。无人工像素级标注，用于阶段2和3的伪标签训练和验证。

## 6. 论文的主要结论与发现

- 渐进式伪标签课程有效：各阶段模型初始mIoU单调递增（0.41→0.47→0.53），证明前一阶段生成的伪标签质量不断提高。

- 公开了最大的核级伪标注TCGA语料库（1,608,061张图块）和模型权重，支持大规模TME研究。

## 7. 优点：方法或实验设计上的亮点

- **创新架构**：UNI2-UperHoVer首次将病理基础模型（ViT-Giant）与双头UperNet解码器结合，同时实现多类语义分割和细胞核实例分离，且通过动态HV合成避免了实例级标注需求，极具实用性。
- **高效标签扩展**：渐进式伪标签课程通过逐阶段提高伪标签质量，无需任何额外人工标注就实现了从7,901张到超160万张图块的跨越，为解决标注瓶颈提供了有效策略。
- **叙事可信性**：TME叙述完全基于可验证的数值特征（如细胞计数、空间熵、距离等），每个临床声称都有对应的量化依据，相比直接像素到文本的生成更可靠。
- **模块化开源**：公开了数据集、模型权重和pip包（`segtme-uni2`），便于社区复现和扩展。
- **定性展示清晰**：IGNITE五张图块的逐行可视化（原图、预测覆盖、分割图、TME token输出）直观展示了框架对多种TME表型的判别力。

## 8. 不足与局限

- **实验不充分**：缺乏外部基准定量比较（如与HoVer-Net、CellPose等在标准数据集上的Dice/ AJI/PQ对比），无消融研究，无统计显著性检验，无病理学家评价。
- **阶段3放大倍率不匹配**：在生成TCGA-UT尺度1-5的伪标签时使用了错误的MPP归一化因子，导致M3在精细形态类（如死亡细胞、上皮细胞）上性能崩溃（IoU近0）。目前只有M2（仅Scale 0）表现可靠。
- **动态HV合成的近似误差**：对于相邻同类别细胞核，连接组件分析可能将多个核误判为一个，从而产生错误的HV目标，影响实例分离准确率。
- **Tile边界拼接问题**：采用最后写入策略的HV图拼接可能在边界引入梯度不连续。
- **规则阈值非最优**：生物学token的数值阈值基于文献和本数据集校准，未通过生存分析等下游任务学习，可能不是最佳。
- **叙事质量未定量评估**：BioNeMo生成叙事的BLEU-4、ROUGE-L、Cohen’s κ和幻觉率均未报告，仅提供定性示例。
- **仅一次运行**：所有结果来自单次训练，缺乏对随机种子方差估计，无法断言结果的可靠性。
- **应用限制**：目前仅验证了路径学框架的可行性，未与临床预后数据关联，实际临床效用未证明。
