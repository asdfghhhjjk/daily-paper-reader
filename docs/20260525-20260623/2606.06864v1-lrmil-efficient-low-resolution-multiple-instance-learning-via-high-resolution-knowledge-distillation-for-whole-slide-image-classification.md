## 摘要
多实例学习（MIL）已成为数字病理学中全切片图像（WSI）分析的标准范式，因为它能够实现无需密集标注的切片级预测。现有的MIL方法通常依赖于高分辨率斑块的详尽提取和编码。然而，这种做法在现实临床环境中存在两个关键限制：它难以在较低放大倍数下捕捉全局视觉线索，并且由于每张切片的大量高分辨率斑块而带来巨大的计算开销。为了解决这些限制，我们提出了一种高效的低分辨率多实例学习（LRMIL）框架，该框架将高分辨率知识转移到低分辨率表示。LRMIL采用两阶段蒸馏策略。首先，斑块级跨分辨率蒸馏将低分辨率斑块嵌入与高分辨率表示对齐。其次，切片级知识蒸馏在切片级监督和教师指导下训练低分辨率学生MIL模型。在推理时，LRMIL仅对低分辨率斑块进行操作，显著减少了数据预处理和计算成本。在多个WSI基准上的大量实验表明，LRMIL在实现更高效推理的同时，始终优于最先进的MIL方法。这些结果突显了LRMIL作为临床病理学中WSI分析的实用且可扩展的解决方案。

## Abstract
Multiple instance learning (MIL) has become a standard paradigm for whole slide image (WSI) analysis in digital pathology, as it enables slide-level prediction without dense annotations. Existing MIL methods typically rely on exhaustive extraction and encoding of high-resolution patches. However, this practice suffers from two critical limitations in real-world clinical settings: it struggles to capture global visual cues at lower magnifications, and incurs substantial computational overhead due to the massive number of high-resolution patches per slide. To address these limitations, we propose an efficient low-resolution multiple instance learning (LRMIL) framework that transfers high-resolution knowledge to low-resolution representations. LRMIL adopts a two-stage distillation strategy. First, patch-level cross-resolution distillation aligns low-resolution patch embeddings with high-resolution representations. Second, slide-level knowledge distillation trains a low-resolution student MIL model under both slide-level supervision and teacher guidance. At inference time, LRMIL operates exclusively on low-resolution patches, substantially reducing data preprocessing and computational cost. Extensive experiments on multiple WSI benchmarks demonstrate that LRMIL consistently outperforms state-of-the-art MIL methods while achieving more efficient inference. These results highlight LRMIL as a practical and scalable solution for WSI analysis in clinical pathology.

### 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：采用**两阶段蒸馏策略**，解耦训练与推理分辨率。训练时利用HR教师模型监督LR学生模型，推理时仅使用LR斑块。
- **第一阶段：斑块级跨分辨率蒸馏**（Patch-level Cross-resolution Distillation）
    - 对同一空间区域，在HR（如20×）和LR（如5×）下提取对应斑块（256×256像素）。一个LR斑块对应K=16个HR斑块（因放大倍数差4倍）。
    - 使用相同的ViT架构编码器：预训练HR编码器f<sub>HR</sub>冻结作为教师，训练LR编码器f<sub>LR</sub>作为学生。
    - 在选定层L（如第0,1,11层），提取HR CLS嵌入与LR对应区域池化特征，通过MSE损失对齐：
      \[
      \mathcal{L}_{\text{layer}}^{(l)} = \frac{1}{K}\sum_{k=1}^K \| \mathbf{z}_{i,r,k,l}^{\text{HR}} - \mathbf{z}_{i,r,k,l}^{\text{LR}} \|_2^2,\quad \mathcal{L}_{\text{patch}} = \sum_{l\in L} \mathcal{L}_{\text{layer}}^{(l)}
      \]
    - 目的：将HR细粒度语义注入LR特征，同时保留LR结构线索。
- **第二阶段：切片级知识蒸馏**（Slide-level Knowledge Distillation）
    - **Step 1**：使用冻结的HR编码器训练一个注意力MIL教师模型，仅用bag-level CE损失（\(\mathcal{L}_{\text{Pre}}^{\text{HR}}\)）。
    - **Step 2**：使用第一阶段蒸馏得到的LR编码器训练LR学生MIL模型，优化目标包含：
        - Bag级CE损失：\(\mathcal{L}_{\text{Pre}}^{\text{LR}} = \text{CE}(\mathbf{p}^{\text{LR}}_i, Y_i)\)
        - Bag级蒸馏：KL散度对齐师生logits（温度T<sub>1</sub>）：\(\mathcal{L}_{\text{bag}} = \text{KL}(\text{softmax}(\mathbf{p}^{\text{HR}}_i/T_1) \parallel \text{softmax}(\mathbf{p}^{\text{LR}}_i/T_1))\)
        - 实例级蒸馏：
            - 软注意力匹配（KL散度，温度T<sub>2</sub>）：\(\mathcal{L}_{\text{inst}}^{\text{soft}} = \text{KL}(\text{softmax}(\tilde{\alpha}^{\text{HR}}_i/T_2) \parallel \text{softmax}(\alpha^{\text{LR}}_i/T_2))\)
            - 硬top-k/bottom-k监督（BCE损失）：将教师注意力最高的k个区域视为正例，最低的k个视为负例，\(\mathcal{L}_{\text{inst}}^{\text{hard}}\)
    - 总损失：\(\mathcal{L}_{\text{slide}} = \mathcal{L}_{\text{Pre}}^{\text{LR}} + \lambda_{\text{bag}}\mathcal{L}_{\text{bag}} + \lambda_{\text{soft}}\mathcal{L}_{\text{inst}}^{\text{soft}} + \lambda_{\text{hard}}\mathcal{L}_{\text{inst}}^{\text{hard}}\)
    - 推理时丢弃教师分支，仅用LR学生模型，无需HR处理。
