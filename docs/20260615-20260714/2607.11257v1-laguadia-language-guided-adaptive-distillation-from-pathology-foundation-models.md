## 摘要
病理基础模型（PFMs）提供了强大的全切片图像（WSI）表示，但其计算成本巨大。虽然知识蒸馏（KD）可以创建高效的学生模型，但现有的多教师方法通常使用次优的统一权重，忽略了组织异质性。我们提出 LaGuadia（语言引导的自适应蒸馏），一个在临床语言指导下动态整合多个 PFM 专业知识来构建紧凑型病理图像编码器的框架。我们的方法采用多阶段流程：首先，从病理报告中提取肉眼可见的临床关键词；其次，通过视觉-语言元教师（MedSigLIP）对齐视觉特征与这些关键词，以提供密集的语义指导；最后，执行自适应 KD，根据教师与临床叙述的语义对齐程度对其贡献进行加权。在 WSI 字幕生成、视觉问答和切片级别分类任务上的实验表明，一个 87M 参数的 LaGuadia 学生模型能够匹配甚至超越 GigaPath 和 UNI 等基础规模模型，实现强大的事实一致性和稳健的泛化能力。这些结果突显了临床语言作为构建高效、可靠数字病理系统的有效语义锚点的作用。代码可在 https://github.com/hvcl/LaGuadia 获取。

## Abstract
Pathology Foundation Models (PFMs) offer powerful Whole Slide Image (WSI) representations but suffer from massive computational costs. While Knowledge Distillation (KD) can create efficient student models, existing multi-teacher methods often use suboptimal uniform weighting that ignores tissue heterogeneity. We propose LaGuadia (Language-Guided Adaptive DistillAtion), a framework that develops a compact pathology image encoder by dynamically integrating expertise from multiple PFMs under clinical linguistic guidance. Our approach utilizes a multi-stage pipeline: first, extracting visually observable clinical keywords from pathology reports; second, aligning visual features with these keywords via a Vision-Language meta-teacher (MedSigLIP) to provide dense semantic guidance; and finally, performing adaptive KD where teacher contributions are weighted based on their semantic alignment with the clinical narrative. Experiments on WSI captioning, visual question answering, and slide-level classification tasks demonstrate that an 87M parameter LaGuadia student model matches or exceeds foundation-scale models such as GigaPath and UNI, achieving strong factual consistency and robust generalization. These results highlight clinical language as an effective semantic anchor for building efficient and reliable digital pathology systems. Code is available at https://github.com/hvcl/LaGuadia.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- 病理基础模型（PFMs，如 GigaPath、UNI、Virchow2）能够为全切片图像（WSI）提供强大的通用表示，但参数量巨大（数亿至十亿级别），推理时计算成本高昂，难以在资源受限的临床环境中部署。
- 知识蒸馏（KD）可将大型教师模型的知识迁移至紧凑的学生模型，但现有的多教师 KD 方法通常使用**统一或简单置信度加权**，忽略了病理组织中强烈的**空间与语义异质性**——不同组织区域可能需要不同专家的知识。
- 仅凭视觉特征难以客观判断哪位教师更可靠，因为相似的组织形态可能对应完全不同的诊断意义，而病理诊断最终以**临床语言报告**形式概括，因此语言可以为教师选择提供更客观的语义参考。
- 本文提出 **LaGuadia（语言引导的自适应蒸馏）**，旨在以临床语言为锚点，动态整合多个 PFM 的专业知识，训练出一个参数少但表达能力强的 WSI 图像编码器。

## 2. 论文提出的方法论

核心思想：通过从病理报告中提取的临床关键词，在共享语义空间中同时对齐视觉与语言，并根据教师模型与临床叙述的语义吻合度**自适应分配蒸馏权重**，从而让轻量学生模型选择性吸收最相关的教师知识。

技术流程分为三个阶段：

### 2.1 关键词提取（Stage 1）
- 基于病理报告，使用 GPT-5-mini 作为关键词提取器，并要求关键词满足三个约束：在 H&E 切片上**可观察**、在 20× 放大倍率下**可识别**、**冗余最小**。
- 对每个病例提取患者专属关键词集 \(K_p\)，并在全数据集上汇总构建**关键词库（Keyword Bank）**，作为后续对比学习中的负样本池。

### 2.2 视觉-语言对齐（Stage 2）
- 以 **MedSigLIP** 作为视觉-语言元教师（Meta-Teacher），其视觉分支 \(\Phi_V\) 和文本分支 \(\Phi_L\) 分别编码图像 patch 和关键词。
- 为处理非肿瘤背景区域，在 \(K_p\) 中加入“无肿瘤（No Tumor Present）”关键词，形成扩展集 \(\hat{K}_p\)。
- 对每个图像 patch \(x\)，计算其与 \(\hat{K}_p\) 中所有关键词的余弦相似度，选出相似度最高的关键词作为**伪标签** \(\hat{K}^*_p\)：
  \[
  \hat{K}^*_p = \arg\max_{k\in\hat{K}_p} \cos(\Phi_V(x), \Phi_L(k))
  \]
- 利用 **SigLIP 损失**进行对比学习，正样本为选中的伪标签，负样本通过**平均相似度阈值**策略从关键词库中动态选取：计算 \(\hat{K}^*_p\) 与 \(K_p\) 内其他关键词的平均余弦相似度作为阈值，低于该阈值的关键词作为负样本。这一步骤旨在增强模态对齐和语义区分能力。

### 2.3 语言引导的自适应知识蒸馏（Stage 3）
- 对每个 patch，将多个教师（GigaPath、UNI、Virchow2）的投影特征与 \(\hat{K}_p\) 中所有关键词的文本嵌入计算相似度，并进行**软投票**（soft voting），选出总得分最高的关键词作为该 patch 的**伪目标关键词** \(\tilde{K}_x\)。
- 根据每个教师视觉特征与 \(\tilde{K}_x\) 文本嵌入的余弦相似度，经 Softmax 计算各教师的**语义加权系数** \(\omega_i(x)\)：
  \[
  \omega_i(x) = \frac{\exp(\cos(\text{Proj}_i(T_{V_i}(x)), \Phi_L(\tilde{K}_x)) / \tau)}{\sum_{j=1}^n \exp(\cos(\text{Proj}_j(T_{V_j}(x)), \Phi_L(\tilde{K}_x)) / \tau)}
  \]
- 蒸馏损失 \(L_{\text{KD}}\) 引导学生模型（ViT-B，86M）的 class token 与加权教师表示尽可能接近：
  \[
  L_{\text{KD}} = \sum_{i=1}^n \omega_i(x) \cdot (1 - \cos(\text{MLP}_i(\text{student}(x)), T_{V_i}(x)))
  \]
- 学生采用 DINOv3 初始化的 ViT-B，并使用 **LoRA** 进行轻量微调。

## 3. 实验设计

- **数据来源**：从 TCGA 的 BRCA、STAD、THCA 三个队列中提取约 2800 万 patch（对应 1910 张 WSI、1801 份病理报告）。关键词库仅由训练/验证集构建，防止测试泄漏。
- **下游任务与基准**：
  - **WSI 字幕生成**：PathText 数据集（BRCA: 974, STAD: 316, THCA: 325），比较不同编码器在固定生成器下的字幕质量。
  - **视觉问答（VQA）**：BRCA 用 WSI‑VQA，STAD 和 THCA 用 WSI‑Bench。
  - **多实例学习（MIL）分类**：在相同 TCGA 队列上执行亚型分类、生存分析、分子分型、进展预测，采用 ABMIL 聚合器。
- **评测指标**：字幕与 VQA 使用 METEOR (M)、BLEU‑4 (B‑4)、ROUGE‑L (R‑L) ，并加入 **Factent** 评估事实一致性；MIL 任务使用 C‑Index（生存分析）和加权 F1（分类）。
- **对比方法**：
  - 大型 PFMs：GigaPath (1.13B)、Virchow2 (631M)、UNI (303M)、MUSK (303M)、MedSigLIP (429M)。
  - 蒸馏基线：单教师 H0‑mini (86M)、多教师 GPFM (303M)。
  - 全部实验采用 5 折交叉验证，报告均值。

## 4. 资源与算力

- 实验使用 **单张 NVIDIA RTX A6000 (48GB)** 完成全部训练。

- Stage 3（自适应蒸馏）优化 150k 迭代，batch size 256，LoRA 部分学习率 \(2\times 10^{-4}\)，蒸馏 MLP 层学习率 \(10^{-3}\)。

## 6. 论文的主要结论与发现

- LaGuadia 仅用 **87M 参数**的学生模型，在 WSI 字幕和 VQA 的**整体评分**上达到或轻微超越 GigaPath (1.13B)、UNI (303M) 等大规模模型，在 STAD、THCA 字幕任务中甚至取得多个单项指标最佳。
- 在 VQA 任务中，LaGuadia 的 **Factent 指标**（事实一致性）显著优于对比方法（例如 BRCA 达 0.9268，STAD 达 0.4737），说明语言引导的蒸馏使模型生成的内容更符合临床事实。
- MIL 分类任务上，LaGuadia 取得最高的整体得分（OVR 0.6538），尤其提升了诊断难度较高的 LAUREN 分型、N 分期以及 THCA 的生存和进展预测，表明语义锚定并未损害视觉特征的判别力，反而有所增强。
- 消融实验证明，单纯的均匀 KD 会降低临床关键指标（如 Factent、生存 C‑Index），而加入语言引导的权重可同时提升语言生成质量和分类性能。

## 7. 优点

- **创新性地引入临床语言作为蒸馏的监督信号**，解决多教师 KD 中权重分配的客观依据问题，比纯视觉启发式更贴合病理诊断逻辑。
- 通过**共识机制和语义相似度加权**，实现 patch 级别的动态教师选择，有效应对组织异质性和语义歧义。
- 学生模型**极轻量（87M）**，训练只需单张 48GB GPU，实用性高。
- 关键词提取时设置了“可观察、可识别、低冗余”的病理约束，并通过加入“无肿瘤”概念处理背景区域，设计细致、贴近真实场景。
- 实验覆盖多种任务和多个癌种，且包含对事实一致性的专项评估，论证严密。
- 使用 LoRA 微调既压缩了可训练参数，又保留了预训练基础，有效提升了训练效率。

## 8. 不足与局限

- 蒸馏仅使用 **class token 级别**的特征对齐，可能丢失 patch 级细粒度形态信息，对需要高空间分辨率的任务（如细胞级检测）可能不够充分。
- 教师权重计算依赖 MedSigLIP 的模态对齐能力，若该元教师在某些病变上的表示有偏差，则会传导至蒸馏过程。
- 关键词提取依赖大语言模型 GPT-5-mini，可能存在**漏提或误提**不可见特征，且未深入讨论不同 LLM 提取器的影响。
- 实验癌种限于 TCGA 的 BRCA、STAD、THCA，对其他器官或罕见肿瘤的泛化性有待验证。
- 训练分两阶段，未进行端到端联合优化，可能限制整体性能上限。
- 未与其他自适应加权策略（如注意力融合、基于不确定性的权重）进行直接比较，说服力略不足。
