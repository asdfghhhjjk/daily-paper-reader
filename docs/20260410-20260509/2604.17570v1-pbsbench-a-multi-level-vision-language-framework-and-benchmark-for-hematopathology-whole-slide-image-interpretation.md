## 摘要
外周血涂片（Peripheral Blood Smear, PBS）是血液病理学中的关键显微检查，能够生成全切片图像（Whole-Slide Imaging, WSI）。与实体组织病理不同，PBS 的解读关注的是单个细胞的形态而非组织结构，因此在视觉特征和诊断推理上具有独特性。然而，当前用于病理学的多模态大语言模型（Multimodal Large Language Models, MLLMs）主要基于实体组织 WSI 开发，在 PBS 任务上表现出较弱的泛化能力。为弥补这一差距，我们构建了 PBSInstr——首个用于 PBS 解读的视觉-语言数据集，包含 353 个 PBS WSI 与显微观察印象段落的配对，以及 2.9 万个带有细胞类型标签和形态描述的细胞级图像裁剪样本。为了促进指令微调，PBSInstr 还包括 2.7 万个细胞裁剪级问答（QA）对及 1,286 个 PBS 切片级 QA 对。在 PBSInstr 的基础上，我们开发了 PBS-VL，这是一个专为血液病理学设计的视觉-语言模型，可在细胞层面和切片层面进行多层次 PBS 解读。为全面评估 PBS 理解能力，我们构建了 PBSBench，这是一项视觉问答（Visual Question Answering, VQA）基准，涵盖四类问题和六项 PBS 解读任务。实验结果显示，PBS-VL 的表现优于现有通用和病理学领域的 MLLM，凸显了 PBS 特定数据的重要价值。我们公开了代码、数据集和模型权重，以推动未来研究。所提出的框架为开发支持血液病理学决策的实用人工智能助手奠定了基础。

## Abstract
Peripheral Blood Smear (PBS) is a critical microscopic examination in hematopathology that yields whole-slide imaging (WSI). Unlike solid tissue pathology, PBS interpretation focuses on individual cell morphologies rather than tissue architecture, making it distinct in both visual characteristics and diagnostic reasoning. However, current multimodal large language models (MLLMs) for pathology are primarily developed on solid-tissue WSIs and struggle to generalize to PBS. To bridge this gap, we construct PBSInstr, the first vision-language dataset for PBS interpretation, comprising 353 PBS WSIs paired with microscopic impression paragraphs and 29k cell-level image crops annotated with cell type labels and morphological descriptions. To facilitate instruction tuning, PBSInstr further includes 27k question-answer (QA) pairs for cell crops and 1,286 QA pairs for PBS slides. Building upon PBSInstr, we develop PBS-VL, a hematopathology-tailored vision-language model for multi-level PBS interpretation at both cell and slide levels. To comprehensively evaluate PBS understanding, we construct PBSBench, a visual question answering (VQA) benchmark featuring four question categories and six PBS interpretation tasks. Experiments show that PBS-VL outperforms existing general-purpose and pathology MLLMs, underscoring the value of PBS-specific data. We release our code, datasets, and model weights to facilitate future research. Our proposed framework lays the foundation for developing practical AI assistants supporting decision-making in hematopathology.

- **研究动机**：

## 二、方法论与技术路线

### 1. 总体框架
论文提出一个三层体系：
1. **PBSInstr 数据集**：多层次视觉语言训练数据。
2. **PBS-VL 模型**：多尺度（cell–patch–slide）视觉语言架构。
3. **PBSBench 基准**：覆盖多任务与问题类型的评测工具集。

### 2. 数据构建：PBSInstr
- 来源：Haemorasis 项目公开的【S-BIAD440】PBS 数据集（353 张全切片，分布于贫血、MDS、对照三类患者）。
- 处理流程：
  1. 切片以 512×512 patch 方式分割并进行质量控制（排除空白、模糊、过密区域）。
  2. 使用 **Cellpose-SAM** 提取细胞实例，并依据形态学特征进行细胞类型分类（Basophil、Eosinophil、Lymphocyte、Monocyte、Neutrophil 等）。
  3. 结合人工与自动标注，生成 **29,673 个带标签的细胞裁剪图像**。
  4. 调用 **GPT‑4o** 生成形态描述、异常特征及诊断提示；通过人审（94% 通过率）控制质量。
  5. 利用这些图像-文本对生成 **QA 样本：27,301 个细胞级问答 + 1,286 个切片级问答**。
- **任务类型（共六类）**：
  - Cell subtyping、Morphology recognition、Abnormality detection、Differential estimation、Diagnosis、Knowledge mastery  
- **问题形式（四种）**：判断题、选择题、填空题、开放问答。

### 3. 模型架构：PBS‑VL
- **核心思想**：模拟病理诊断的层次性，从“单细胞 → patch → 全片”逐级推理。  
- **三层结构**：
  1. **细胞层（Cell-level）**
     - 采用 **BLIP‑2 框架**：冻结视觉编码器（DinoBloom ViT‑L/14），引入学习式 Q‑former 产生视觉语言 token。
  2. **细胞–patch 对齐**
     - 使用 **Perceiver 模块** 压缩 patch 特征，与 Q‑former 输出进行全局/局部对齐：
       - 全局损失：最小化 patch–cell token 均方差；
       - 局部损失：基于对比学习最大化正配对相似度。
  3. **切片层（Slide-level）**
     - 利用第二个 Perceiver Resampler 聚合 patch token → slide token；
     - 连接冻结的视觉语言模型（VLM，基于 Qwen2.5‑VL‑7B），通过 instruction tuning 进行切片级问答微调。
- **训练分阶段**：
  1. 表征学习（BLIP‑2 损失：对比+匹配+语言建模）  
  2. Cell‑Patch 对齐（仅训练 Perceiver）  
  3. 细胞级指令微调  
  4. 切片级指令微调

## 六、主要结论与发现

- **总体性能**：PBS‑VL 在所有问题类型与任务上均显著优于通用及现有病理 MLLMs。
- **泛化能力**：即使在外部数据集（AML‑LMU、APL、LISC）上仍保持领先。
- **任务能力差异**：
  - 通用模型在选择题（带提示）上表现好，但在需视觉逻辑推理的开问题中显著下降；
  - PBS‑VL 更擅长从视觉线索中提取形态特征并与医学概念对齐。
- **关键改进来源**：
  - PBS 特定数据对模型视觉对齐至关重要；
  - Cell–Patch 对齐机制减少显式分割负担且保留细胞语义。
