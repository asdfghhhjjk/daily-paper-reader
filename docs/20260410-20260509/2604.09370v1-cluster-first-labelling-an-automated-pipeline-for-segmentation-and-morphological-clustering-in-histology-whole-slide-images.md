## 摘要
在组织学全切片图像（WSIs）中标注组织成分是一项极度耗费人力的工作：单个切片可能包含成千上万个结构——细胞、细胞核及其他形态上可区分的对象——每一个都需要手工绘制边界并进行分类。我们提出了一种基于“先聚类”的云原生端到端自动化流程，以简化这一过程。该系统对 WSIs 进行切片，过滤掉不太可能含有有效信息的切片，使用 Cellpose-SAM 对组织成分（包括细胞、细胞核及其他形态相似的结构）进行分割，借助预训练的 ResNet-50 提取神经嵌入，使用 UMAP 进行降维，并通过 DBSCAN 聚类将形态相似的对象进行分组。在这一范式下，人类标注者仅需对代表性簇进行标注，而非单个对象，从而将标注负担降低数个量级。我们在来自三种物种（人类、鼠类、兔类）共13种不同组织类型的3696个组织成分上对该流程进行评估，通过每片切片的匈牙利算法匹配来衡量无监督聚类与独立人工标签的一致性。系统实现了加权簇标签对齐准确率96.8%，其中13种组织类型中的7种达到了完全一致。该流程、配套标注网页应用及所有评估代码均作为开源软件发布。

## Abstract
Labelling tissue components in histology whole slide images (WSIs) is prohibitively labour-intensive: a single slide may contain tens of thousands of structures--cells, nuclei, and other morphologically distinct objects--each requiring manual boundary delineation and classification. We present a cloudnative, end-to-end pipeline that automates this process through a cluster-first paradigm. Our system tiles WSIs, filters out tiles deemed unlikely to contain valuable information, segments tissue components with Cellpose-SAM (including cells, nuclei, and other morphologically similar structures), extracts neural embeddings via a pretrained ResNet-50, reduces dimensionality with UMAP, and groups morphologically similar objects using DBSCAN clustering. Under this paradigm, a human annotator labels representative clusters rather than individual objects, reducing annotation effort by orders of magnitude. We evaluate the pipeline on 3,696 tissue components across 13 diverse tissue types from three species (human, rat, rabbit), measuring how well unsupervised clusters align with independent human labels via per-tile Hungarian-algorithm matching. Our system achieves a weighted cluster-label alignment accuracy of 96.8%, with 7 of 13 tissue types reaching perfect agreement. The pipeline, a companion labelling web application, and all evaluation code are released as open-source software.

## 一、核心问题与研究背景

- **研究动机**：在组织病理学中，全切片图像（Whole Slide Images, WSIs）的细胞标注过程极度耗时。单张高分辨率切片可能包含数万至十万的细胞或细胞核，现有人工逐一描绘边界并分类的方法不仅劳动密集，还限制了大规模数据集的建立。
- **研究背景**：尽管已有深度学习分割模型（如 U-Net、Hover-Net、Cellpose）可准确分割细胞，但后续的人工标注工作仍需逐实例操作。本文提出一种“**先聚类、后标注（Cluster-First Labelling）**”的新范式，旨在显著减少专家标注成本。
- **核心问题**：能否通过无监督聚类将形态相似的组织成分归组，从而只需对代表性簇进行一次人工标注，就能实现高一致性与高效率的自动标注？

## 二、方法论：核心思想与技术细节

### 1. 核心思想
- **理念**：以聚类为标注前步骤，通过自动分割与聚类，使人工仅需标注几类代表性群组。
- **目标**：将标注任务从 \(O(N)\)（逐细胞）降至 \(O(K)\)（逐簇），其中 \(K \ll N\)。

### 2. 流程概述（Pipeline Architecture）

该系统为 **云原生端到端自动化管线**，由若干独立的容器化模块组成：

1. **WSI 切片与质量过滤**  
   - 使用 OpenSlide 将原始 `.ndpi` 格式的切片分割为 **512×512 像素瓦片（tile）**。  
   - 基于六个指标执行质量过滤：边缘密度、亮 / 暗像素比、强度标准差、拉普拉斯方差（聚焦度）、颜色通道方差等。  
   - 去除无信息或模糊瓦片以减少计算量。

2. **细胞分割（Cell Segmentation）**  
   - 调用 **Cellpose-SAM** 模型：结合 Cellpose 的流场分割机制与 Segment Anything 编码器，无需重训练即可跨组织泛化。  
   - 参数：flow threshold = 0.4, cell-prob threshold = 0.0。  
   - 输出每个对象的边界 mask、bounding box、polygon。

3. **特征提取与降维（Embedding & Projection）**  
   - 每个分割对象的图像裁剪后输入 **ResNet-50（ImageNet 预训练）** 提取 2048 维特征。  
   - 使用 **UMAP** 降维至 50 维，保留局部与全局结构：

     \[
     \text{UMAP}(X) \rightarrow Z_{50} \quad\text{(k=15, min\_dist=0.1)}
     \]

4. **无监督聚类（DBSCAN Clustering）**  
   - **密度聚类算法 DBSCAN**: 自动估计邻域半径 ε 并设定 `min_samples = 5`。  
   - 通过 RAPIDS cuML GPU 加速实现。  
   - 噪声点标记为 cluster −1。

5. **人机交互标注与评估（Human Annotation & Evaluation）**  
   - 使用 FastAPI 构建前端标注应用，实现交互标签分配与匈牙利算法对齐。
   - **匈牙利算法（Hungarian Algorithm）公式描述**：

     对模型簇与人工标签间的匹配矩阵 \(C_{ij}\)，最大化一致性：

     \[
     \pi^* = \arg\max_\pi \sum_i C_{i, \pi(i)}
     \]

     计算对齐准确率作为评估指标。

6. **云端执行架构**  
   - 基于 Azure ML 平台，支持 **多 GPU 节点并行执行**（slide-level granularity）。
