## 摘要
近年来，计算机视觉技术取得了显著进展，并在医学领域中展现出了多样而深远的应用。在本文中，我们提出了一个用于在宫颈涂片图像中检测 Bethesda 细胞的新框架，该框架是为国际生物医学影像研讨会（ISBI）联合举办的 Riva Cytology Challenge 的 Track B 开发的。本研究专注于提升用于细胞检测的计算机视觉模型，其性能通过 mAP50-95 指标进行评估。我们提出的解决方案基于 YOLO 和 U-Net 架构的集成模型，随后采用重叠去除技术和二分类器进行精细化处理。在比赛中，我们的框架以 0.5909 的 mAP50-95 分数获得第二名。实现代码已公开在以下仓库：github.com/martinamster/riva-trackb。

## Abstract
Computer vision techniques have advanced significantly in recent years, finding diverse and impactful applications within the medical field. In this paper, we introduce a new framework for the detection of Bethesda cells in Pap smear images, developed for Track B of the Riva Cytology Challenge held in association with the International Symposium on Biomedical Imaging (ISBI). This work focuses on enhancing computer vision models for cell detection, with performance evaluated using the mAP50-95 metric. We propose a solution based on an ensemble of YOLO and U-Net architectures, followed by a refinement stage utilizing overlap removal techniques and a binary classifier. Our framework achieved second place with a mAP50-95 score of 0.5909 in the competition. The implementation and source code are available at the following repository: github.com/martinamster/riva-trackb

## 二、方法论与技术结构

### 1. 总体框架
- 构建 **三阶段优化管线**：  
  (1) YOLO 模型训练与集成  
  (2) U-Net 模型补充高精度检测  
  (3) 后处理优化流程（去重与分类修正）

### 2. 核心技术思想
- **YOLO 检测阶段**：  
  - 改进原始数据中固定为 100×100 像素的标注框，调整为 10×10 至 120×120 的多尺度框，增强对不同形态细胞的定位能力。  
  - 使用 **YOLOv8 Nano/Small/Medium** 版本，训练过程中采用 1024×1024 输入图像与批大小 16。  
  - 通过 **中心距离阈值**（12像素）与置信度（0.35）实现模型间预测融合。

- **U-Net 分割阶段**：  
  - 采用 **ResNet34 作为骨干网络**，以热力图回归方式预测细胞中心。  
  - 使用 2D 高斯核表示注释目标（σ=1/6标注尺寸），以 MSE 损失优化。  
  - 多尺度推理（0.8×、1.0×、1.2×）并使用 **25×25 最大池化**提取局部最大值。

- **模型集成策略**：  
  - 首先融合两种 YOLO 模型（20×20 与 50×50 尺度）构成初级集成；  
  - 再将 YOLO 集成结果与 U-Net 输出进行二次融合，采用中心距离相同阈值（12像素）；  
  - 所有单独预测（未匹配到另一模型）也全部保留，以避免漏检。

### 3. 后处理优化流程（三阶段）

1. **非极大值抑制（NMS）**：
   - IoU 阈值设定为 0.75（偏宽松），保留紧邻但不同目标，兼顾高密度细胞情况下的准确定位。
2. **空间密度过滤**：
   - 将图像划分为 4×4 网格区域；  
   - 高密度区域（>30检测）采用置信度阈值 0.1；  
   - 低密度区域（<30检测）放宽阈值至 0.001。
3. **二分类精炼阶段**：
   - 采用 **EfficientNet-B0 分类器**区分“真实细胞”与“背景伪影”。  
   - 训练集通过 **hard negative mining**生成，损失函数为加权二元交叉熵。  
   - 推理阶段使用置信度阈值 0.05，以保证高灵敏度。
