## 摘要
数字病理组织切片分割面临的一个关键挑战是染色间及染色内的差异，这会导致模型性能下降。由于为每种染色方式进行标注既昂贵又耗时，研究者提出了利用CycleGAN进行染色迁移的方法，以用单一染色的标注来训练多染色分割模型。然而，由于某些染色对之间存在“一对多”的映射关系，CycleGAN在图像转换过程中容易引入噪声，并与其循环一致性损失产生冲突。为此，我们提出了Domain Shift Aware CycleGAN（DSA-CycleGAN），以减少此类噪声的产生。此外，我们还评估了机器学习领域中针对类似问题提出的若干改进方法，并在多染色肾小球分割任务中将其与DSA-CycleGAN的效果进行了比较。实验结果表明，DSA-CycleGAN不仅提升了肾小球分割的性能，还在降噪方面优于其他方法，尤其在生物学差异较大的染色之间转换时效果更加显著。相关代码已公开于：https://github.com/zeeshannisar/DSA-CycleGAN。

## Abstract
A key challenge in segmentation in digital histopathology is inter- and intra-stain variations as it reduces model performance. Labelling each stain is expensive and time-consuming so methods using stain transfer via CycleGAN, have been developed for training multi-stain segmentation models using labels from a single stain. Nevertheless, CycleGAN tends to introduce noise during translation because of the one-to-many nature of some stain pairs, which conflicts with its cycle consistency loss. To address this, we propose the Domain Shift Aware CycleGAN, which reduces the presence of such noise. Furthermore, we evaluate several advances from the field of machine learning aimed at resolving similar problems and compare their effectiveness against DSA-CycleGAN in the context of multi-stain glomeruli segmentation. Experiments demonstrate that DSA-CycleGAN not only improves segmentation performance in glomeruli segmentation but also outperforms other methods in reducing noise. This is particularly evident when translating between biologically distinct stains. The code is publicly available at https://github.com/zeeshannisar/DSA-CycleGAN.

## 一、研究背景与核心问题

- **研究领域**：数字病理学中的组织图像分割，特别是肾脏组织中的肾小球分割任务。  
- **问题背景**：
  - 病理切片常采用不同染色方法（如PAS、Jones H&E、Sirius Red、CD34、CD68等），但染色方式会导致图像外观差异（即“染色间差异”）。
  - 同一染色内部也存在实验室、设备或个体差异引起的“染色内差异”。
  - 这些非一致性会造成模型在跨染色域上的性能急剧下降。
- **现实困难**：
  - 为每种染色图像建立标注集非常昂贵且耗时。
  - 因此，业界采用无监督的染色迁移（stain transfer）方法，如 **CycleGAN**，以实现多染色学习。
- **现有方法的缺陷**：
  - CycleGAN在某些染色之间（一对多的映射）转换时，会因循环一致性约束与影像内容冲突而引入噪声。
  - 该噪声导致“域偏移（Domain Shift）”，从而降低下游分割模型性能。
- **核心目标**：
  - 减少CycleGAN中由领域偏移引起的转换噪声；
  - 提升跨染色分割模型的鲁棒性，尤其在生物学差异大的染色之间。

## 二、方法论：DSA-CycleGAN 架构与原理

### 1. 核心思想
- 基于CycleGAN框架，新增**领域偏移感知损失（L<sub>dsl</sub>）**；
- 利用“**领域偏移度量指标（Domain Shift Metric, DSM）**”作为定量指导，使生成的图像在特征分布上更接近源域；
- 无需额外的分割掩膜或标注信息，可自监督优化。

### 2. 技术机制
- DSM通过预训练的源域分割网络计算不同域图像的特征分布差异：
  - 在每层卷积滤波器激活下计算均值特征分布；
  - 使用Wasserstein距离衡量源域分布与翻译域分布的差异；
  - 公式表示为：
    \[
    DSM(p_S, p_{S'}) = \frac{1}{k} \sum_{i=1}^{k} W(p_{sc_i}, p_{s'c_i})
    \]
- 将该指标作为附加损失整合进CycleGAN的总体目标函数：
  \[
  L_{DSA-CycleGAN} = L_{adv} + w_{cyc}L_{cyc} + w_{id}L_{id} + w_{dsl}L_{dsl}
  \]
  其中L<sub>dsl</sub>由三个部分构成：
  - L<sub>dsl fake</sub>：目标到源（T→S）翻译；
  - L<sub>dsl cyc</sub>：循环一致性重建；
  - L<sub>dsl id</sub>：源域恒等映射。

### 3. 与其他方法的区别
- 不依赖显式分割掩膜（区别于自监督法）；
- 可同时约束多个子路径（相比只在生成端约束的替代方法更全面）；
- 设计灵活，可扩展至 StarGAN 或 Encoder-Decoder 等框架。

## 三、实验设计与评估体系

### 1. 数据集与任务场景
- 来自 **汉诺威医学院（Hannover Medical School）** 的肾组织样本：
  - 共10名患者；
  - 包含五类染色：PAS（源域）、Jones H&E、Sirius Red、CD68、CD34。
- 组织切片由自动染色器（Ventana Benchmark Ultra）制备，40×放大扫描。
- 专家通过 Cytomine 进行肾小球标注。

### 2. 数据划分与数量

| CD34 | - | 595 | 1019 |  
| CD68 | - | 521 | 1046 |

### 3. 实验流程
1. 在 PAS 染色上训练 U-Net 分割模型（source segmentation）。  
2. 使用不同CycleGAN模型将目标染色图像转为PAS域（T→PAS）。  
3. 将训练好的 U-Net 直接应用于这些翻译后的图像，评估其分割性能。

### 4. 对比基线
- 原始 **CycleGAN (baseline)**  
- **CycleGAN w/Gaussian-noise**（添加噪声扰动）  
- **CycleGAN w/Extra-channels**（将噪声导入额外通道）  
- **CycleGAN w/Self-supervision**（基于自监督一致性）  
- **DSA-CycleGAN** 及其三个消融版本（仅dsl fake、仅dsl cyc+id、完整模型）

### 5. 评估指标
- 下游分割任务的 **F1-score**；
- 比较 UNet(PAS) 模型在各种转换图像上的性能；
- 统计显著性：p < 0.05。

## 六、主要结论与发现

- DSA-CycleGAN 能有效降低CycleGAN翻译中产生的噪声；
- **在信息差异较大（如HC↔IHC）染色之间**表现出显著优势；
- 下游分割性能（F1-score）在所有染色类别上优于标准CycleGAN；

- 消融实验显示：需要同时使用所有L<sub>dsl</sub>分量，才能达到最优性能。
