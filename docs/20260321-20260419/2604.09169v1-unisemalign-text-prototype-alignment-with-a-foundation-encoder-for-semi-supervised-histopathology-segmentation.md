## 摘要
在计算病理学中，半监督语义分割仍然具有挑战性，因为像素级标注稀缺且伪标签监督不可靠。我们提出了 UniSemAlign，这是一种双模态语义对齐框架，通过将显式的类别级结构注入像素级学习，来增强视觉分割性能。该方法建立在经过病理学预训练的 Transformer 编码器之上，UniSemAlign 在共享的嵌入空间中引入了互补的原型级和文本级对齐分支，提供结构化的指导，从而减少类别歧义并稳定伪标签的优化。对齐后的表示与视觉预测融合，为未标注的病理组织切片图像生成更可靠的监督信号。该框架通过有监督分割、跨视角一致性以及跨模态对齐目标进行端到端训练。在 GlaS 和 CRAG 数据集上的大量实验表明，UniSemAlign 在有限监督条件下显著优于近期的半监督基线模型，在仅使用 10% 标注数据的情况下，Dice 系数在 GlaS 上提升最多可达 2.6%，在 CRAG 上提升最多可达 8.6%，并且在 20% 监督下也取得了显著改进。代码可在：https://github.com/thailevann/UniSemAlign 获取。

## Abstract
Semi-supervised semantic segmentation in computational pathology remains challenging due to scarce pixel-level annotations and unreliable pseudo-label supervision. We propose UniSemAlign, a dual-modal semantic alignment framework that enhances visual segmentation by injecting explicit class-level structure into pixel-wise learning. Built upon a pathology-pretrained Transformer encoder, UniSemAlign introduces complementary prototype-level and text-level alignment branches in a shared embedding space, providing structured guidance that reduces class ambiguity and stabilizes pseudo-label refinement. The aligned representations are fused with visual predictions to generate more reliable supervision for unlabeled histopathology images. The framework is trained end-to-end with supervised segmentation, cross-view consistency, and cross-modal alignment objectives. Extensive experiments on the GlaS and CRAG datasets demonstrate that UniSemAlign substantially outperforms recent semi-supervised baselines under limited supervision, achieving Dice improvements of up to 2.6% on GlaS and 8.6% on CRAG with only 10% labeled data, and strong improvements at 20% supervision. Code is available at: https://github.com/thailevann/UniSemAlign

- **总体目标**

## 二、方法论：UniSemAlign 框架

### 核心思想

- 在病理预训练 Transformer 编码器（UNI ViT-B/16）基础上，引入**双模态语义对齐机制**：
  - **原型级对齐（Prototype Alignment）**：学习每类的可训练原型向量，作为语义锚点；
  - **文本级对齐（Text Alignment）**：通过冻结的 CONCH 文本编码器生成基于可学习上下文 token 的类语义嵌入。  
- 将两者与像素嵌入在统一空间的余弦相似度进行计算，从而提供类层级语义约束。

### 关键技术细节（文字版算法流程）

1. **视觉编码与解码**  
   - 使用 UNI ViT-B/16 提取多层特征（低层 F2、高层 F9）。  
   - 通过 DeepLabV3+ 解码器输出初步分割预测（decoder logits）。  

2. **像素嵌入投影**  
   - 高层特征经 3×3 卷积映射至共享语义空间 (D=256)，并 L2 归一化。  

3. **原型分支**  
   - 学习 C 个类原型 \( P \in \mathbb{R}^{C \times D} \)。  
   - 计算像素与原型间的余弦相似度生成对齐 logits。  

4. **文本分支**  
   - 使用 CoOp 提示学习 (learnable context tokens)，嵌入类名后以冻结的 CONCH 文本编码器生成每类文本嵌入 \( T \in \mathbb{R}^{C \times D} \)。  
   - 再与像素嵌入计算余弦相似度。  

5. **伪标签融合与生成**  
   - 对齐结果与解码器 logits 融合：  
     \[
     S_{\text{fuse}} = S_{dl} + \eta_p S_{zp} + \eta_t S_{zt}
     \]
     其中 \(\eta_p, \eta_t\) 为加权系数（文中均设为 0.1）。  
   - 对 unlabeled 样本采用 CorrMatch 风格的弱—强一致性学习，通过 softmax + 置信度阈值生成伪标签。

6. **损失函数**  
   - **监督部分**：三分支交叉熵损失 + 原型与文本嵌入间的一致性约束（MSE 对齐损失）。  
   - **无监督部分**：弱—强一致性下的三项损失（Hard CE、Soft KL、一致性 Correlation）。  
   - 总损失：
     \[
     L = \frac{1}{2}(L_{\text{sup}} + L_{\text{unsup}})
     \]

## 六、主要结论与发现

- UniSemAlign 显著提升半监督分割性能：  
  - **GlaS**：mDice 提升最高 +2.6%，mJaccard +4%。

- 原型分支改善结构连贯性，文本分支抑制错误激活；两者互补。
