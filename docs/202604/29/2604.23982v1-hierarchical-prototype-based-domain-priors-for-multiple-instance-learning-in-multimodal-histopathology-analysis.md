## 摘要
数字病理学通过实现对千兆像素级全切片图像（Whole Slide Images，WSIs）的计算分析，彻底改变了诊断工作流程。然而，要有效解析其复杂的肿瘤微环境仍是一项艰巨的挑战。现有的多实例学习（Multiple Instance Learning, MIL）框架通常将全切片图像视为无结构的补丁集合，从而忽略了关键的形态语义与空间几何信息。这种缺乏诱导偏置的做法往往导致在背景噪声上过拟合，并无法使视觉特征与高层诊断知识对齐。为克服这些限制，我们提出层次化原型域先验（Hierarchical Prototype-based Domain Priors, HPDP）框架，这是一种用于联合组织病理学诊断与预测的统一多模态方法。HPDP通过引入形态锚定原型系统（Morphologically Anchored Prototype System, MAPS），将学习过程锚定于可解释的形态聚类；并利用正弦位置编码器（Sinusoidal Positional Encoder, SPE）显式建模组织结构，从而缓解数据驱动的“黑箱”问题。此外，我们通过分层跨模态对齐（Hierarchical Cross-Modal Alignment, HCMA）模块弥合语义鸿沟，使用由大型语言模型（Large Language Model, LLM）生成的描述语境化优化视觉表征。针对七个癌症队列的大量实验证明，HPDP在性能、鲁棒性与可解释性方面始终取得了最先进的表现。

## Abstract
Digital pathology has fundamentally altered diagnostic workflows by enabling the computational analysis of gigapixel Whole Slide Images (WSIs), yet effectively deciphering their complex tumor microenvironments remains a formidable challenge. Existing Multiple Instance Learning (MIL) frameworks typically treat Whole Slide Images as unstructured bags of patches, discarding critical morphological semantics and spatial geometry. This lack of inductive bias often leads to overfitting on background noise and fails to align visual features with high-level diagnostic knowledge. To overcome these limitations, we propose the Hierarchical Prototype-based Domain Priors (HPDP) framework, a unified multimodal approach for joint histopathology diagnosis and prognosis. HPDP mitigates the data-driven "black box" issue by introducing a Morphologically Anchored Prototype System (MAPS), which anchors learning to interpretable morphological clusters, and a Sinusoidal Positional Encoder (SPE) to explicitly model tissue architecture. Furthermore, we bridge the semantic gap via a Hierarchical Cross-Modal Alignment (HCMA) module, using Large Language Model (LLM)-generated descriptions to contextually refine visual representations. Extensive experiments across seven cancer cohorts demonstrate that HPDP consistently achieves state-of-the-art performance with superior robustness and interpretability.

## 二、方法论与技术框架（HPDP）

### 1. 总体思想
HPDP（Hierarchical Prototype-based Domain Priors）是一个多模态端到端学习框架，它通过**形态学原型先验 + 空间位置编码 + 语言语义引导**实现跨模态特征融合。其总体目标是使模型在结构上“有生物学的约束”、在语义上“有临床知识对齐”。

### 2. 模型架构概述
HPDP 包含三个关键模块：

#### (1) Morphologically Anchored Prototype System (MAPS)
- **核心作用**：在特征聚合中引入形态学原型作为域先验。  
- **设计逻辑**：
  - **Prior Experts**（先验专家）：基于 K-means 聚类的组织形态中心（如肿瘤、基质、血管等）。这些专家在训练中受固定域指导约束。
  - **Adaptive Experts**（自适应专家）：无监督学习参数，用于捕获任务特异性的细粒度特征。  
- **关键思想**：通过余弦相似与点积相结合的双路径机制分别建模语义方向与特征强度，从而获得既稳定又具有任务敏感性的表征。

#### (2) Sinusoidal Positional Encoder (SPE)
- **目的**：显式建模组织空间结构。  
- **方法**：将每个补丁的 (x, y) 坐标映射到高维正弦/余弦编码，形成可学习的空间嵌入，以恢复WSI中的几何关系。

#### (3) Hierarchical Cross-Modal Alignment (HCMA)
- **目标**：将视觉表征与由 LLM 生成的临床描述进行语义对齐。  
- **两阶段策略**：
  1. **全局语义调制 (Global Semantic Modulation)**：根据文本特征生成缩放与偏移参数，对视觉通道进行逐特征加权。
  2. **结构化上下文传播 (Structural Context Propagation)**：通过多头注意力机制在视觉与语义空间中进行双向传播校准，实现视觉原型与语言描述的深度融合。

### 3. 训练目标与损失
- 任务损失：  
  - 分类任务采用交叉熵损失。
