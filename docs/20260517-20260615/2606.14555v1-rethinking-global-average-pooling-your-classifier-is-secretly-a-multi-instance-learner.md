## 摘要
现代图像分类器广泛采用全局平均池化（GAP）后接线性分类头。这种线性性保证了图像级 logits 等于在 GAP 之前逐点将分类头应用于特征网格所获得的 logits 的平均值。因此，标准分类器可能固有地保留空间类别证据，即使图像级预测不正确时这些证据仍然可恢复。这种结构自然地暗示了一种多实例学习（MIL）解释，其中图像被视为空间实例的包。在这个公式中，我们证明每张图像仅用一个标签训练的标准分类器仍然可以在多目标场景中学习到预期的分类任务。我们进一步利用这一特性将图像级 logits 分解为预测网格，提供一种事后诊断方法，以提取 GAP 否则会掩盖的空间类别证据。我们的系统评估显示，现成模型一致地恢复了前景区域内的真实类别。MIL 解释进一步表明，常见的分类器失败反映了均值聚合的已知局限性。

## Abstract
Modern image classifiers widely adopt global average pooling (GAP) followed by a linear classification head. This linearity ensures that the image-level logits equal the average of logits obtained by applying the classification head pointwise to the feature grid prior to GAP. Consequently, standard classifiers may inherently retain spatial class evidence that remains recoverable even when the image-level prediction is incorrect. This structure naturally suggests a multiple-instance learning (MIL) interpretation, where an image is viewed as a bag of spatial instances. Within this formulation, we demonstrate that standard classifiers trained with a single label per image can still learn the intended classification task in multi-object scenes. We further exploit this property to decompose image-level logits into a prediction grid, providing a post-hoc diagnostic to extract spatial class evidence that GAP otherwise obscures. Our systematic evaluation reveals that off-the-shelf models consistently recover the ground-truth class within foreground regions. The MIL interpretation further suggests that common classifier failures reflect known limitations of mean aggregation.

### 6. 论文的主要结论与发现

1. **密集读数暴露空间类证据**：在 ImageNet-1K 上，所有评估模型的前景检测率（FG Det.）均超过 90%，局部定位准确率（Loc.）从 33.7% 到 77.3% 不等，表明分类器在前景区域成功识别了目标类别，即使图像级预测可能错误。
2. **图像级失败源于聚合**：在 ImageNet-A 上，图像级准确率大幅下降（如 EfficientNet 仅 3.8%），但空间检测率（Det.）仍达 30.6%~56.9%，说明局部类证据被 GAP 稀释，而非特征缺失。
3. **单标签监督可学得多对象定位**：合成实验显示，仅用单标签训练，ResNet-18 的局部定位准确率达 94%，ViT-S/32 达 63%（受限于训练配方），说明级联平均池化能够从多实例包中学习。
4. **架构差异**：ViT 模型在未遮挡图像上背景激活率更高（约 90%），但前景遮挡后大幅下降，反映自注意力扩散了物体证据；CNN 背景激活率较低且遮挡后变化较小，甚至部分类别出现反直觉的上升（表 8），这可能与卷积感受野内的特征混合有关。
5. **上下文依赖与捷径**：部分类别（如“网站”、“篮球”）即使在遮挡前景后，背景仍几乎 100% 预测正确，表明模型学习了上下文捷径；另一部分类别（如“jacamare”）在遮挡前背景激活率低，遮挡后反而升高，暗示模型在遮挡边界处的异常响应。
6. **MIL 视角提供新解释**：分类器的失败（如多标签丢失、对抗攻击成功）可归因于均值聚合对稀疏/竞争证据的抑制，这与 MIL 中均值池化的已知局限性一致。

### 8. 不足与局限

- **空间分辨率低**：依赖最终特征网格，通常为 7×7 或 14×14，无法精确到像素级边界，限制了定位精度。
- **缺乏背景类**：标准分类器无显式背景类，导致背景区域被强制映射到前景类别，产生不可靠的预测；阈值化无法完全消除这一问题。
- **独立性假设近似**：将空间特征向量视为独立实例忽略了 CNN 的重叠感受野和 ViT 的自注意力传播，尤其 ViT 中背景 token 会携带来自前景的全局信息，使得“背景激活”部分来自扩散而非真实空间支持。
- **遮挡引入分布偏移**：使用黑色块或均值填充遮挡会引入伪影，尤其对 CNN 可能触发异常行为（如遮挡后背景激活率反升），需谨慎解释。
- **缺少与现有方法的定量比较**：未与 CAM、Grad-CAM、WSOL 等方法进行指标对比（如定位精确度、点定位 IoU 等），无法直接评判其诊断质量相对优劣。
- **计算资源信息缺失**：未报告 GPU 型号、训练时间，不利于复现和评估实用性。
- **非线性头与 [CLS] 聚合扩展不完整**：对于 MLP 头或 [CLS] token 的 ViT，逐点读数不再精确等于图像级 logits 的分解，解释性下降，实验仅初步验证了部分场景。
