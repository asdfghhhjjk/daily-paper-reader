## 摘要
传统全切片图像（WSI）分析方法通常依赖多实例学习（MIL）范式，即在放大倍数下提取补丁级特征，并将其聚合以进行切片级预测。然而，这种详尽的补丁级处理计算成本高昂，严重限制了WSI分析的效率和可扩展性。为应对这一挑战，我们提出了PathCTM（面向病理学的连续思考模型），该模型能够实现十亿像素WSI的令牌高效尺度空间连续推理。PathCTM将诊断推断构建为动态的序列信息追踪，逐步从低倍全局观察过渡到高倍局部检查，并在积累足够证据以有效约束决策不确定性时自适应终止推理。具体而言，它采用条件计算进行动态尺度切换，并辅以注意力引导的区域剪枝以及置信度感知的提前终止。大量实验表明，与基于MIL的标准方法相比，PathCTM将所需图像补丁数量减少了95.95%，推理时间缩短约95.62%，同时AUC保持不变。代码已开源：https://github.com/JSGe-AI/PathCTM。

## Abstract
Traditional whole slide image (WSI) analysis methods typically rely on the multiple instance learning (MIL) paradigm, which extracts patch-level features at high magnification and aggregates them for slide-level prediction. However, such exhaustive patch-level processing is computationally expensive, severely limiting the efficiency and scalability of WSI analysis. To address this challenge, we propose PathCTM (a Pathology-oriented Continuous Thought Model) that enables token-efficient scale-space continuous reasoning for gigapixel WSIs. PathCTM formulates diagnostic inference as a dynamic sequential information pursuit. It progressively transitions from low-magnification global to high-magnification local inspection, and adaptively terminates inference when sufficient evidence is gathered to effectively bound decision uncertainty. Specifically, it uses conditional computation for dynamic scale switching with attention-guided region pruning, coupled with confidence-aware early stopping. Extensive experiments demonstrate that, compared with standard MIL-based methods, PathCTM reduces the number of required image patches by 95.95% and shortens inference time by approximately 95.62%, while maintaining AUC without degradation. Code is available at https://github.com/JSGe-AI/PathCTM.

### 2. 方法论

**核心思想**：PathCTM将WSI分析重新定义为**动态连续尺度空间推理**，融合了“时间维度思考”（CTM的内部迭代）与“尺度维度思考”（多分辨率层次推理）。形式化地，诊断是一个最小化条件熵 $H(Y|Z_t)$ 的序列信息获取过程，其中 $Z_t$ 为时步 $t$ 的隐状态。

**关键技术细节**：

- **尺度空间连续推理**：模型在低倍率下进行 $n$ 步连续推理，维持一个带有FIFO机制的隐状态历史。当推理步数上限达到但置信度不足时，系统触发多尺度动态切换，进入更高分辨率的尺度的局部推理。跨尺度信息通过**同步融合**保持：将当前精细尺度的输出与上一尺度最置信的输出拼接作为最终预测。
  
- **注意力引导区域剪枝 (Attention-Guided Region Pruning)**：提出用注意力分数作为单个图像块对决策影响的一阶代理（Proposition 1），选择当前推理轨迹中最置信时刻的前 $K$ 个高注意力块作为下一尺度的候选区域。这等价于求解信息增益最大化子集的选择问题（公式6），使计算复杂度从 $O(N)$ 降至 $O(K)$。同时引入**联合注意力校核**机制，在进入更高尺度后重新计算注意力权重以过滤噪声候选，阻断错误传播。

- **置信度早停 (Confidence-Aware Early Stopping)**：定义置信度 $C_{t,L} = 1 - \frac{H(\hat{y}_{t,L})}{\log N}$，其中 $H$ 为熵。当 $C_{t,L} \geq \delta$ 时立即终止推理。该策略基于Fano不等式（Proposition 2）：高条件熵意味着不可忽略的误差下界，从而为早停提供信息论依据。这使得模型能根据实例难度自适应调整推理深度（易案早停、难案深推）。

**训练损失**：采用复合目标函数，同时对每个尺度的最小损失时刻和最大置信度时刻计算交叉熵损失，均衡准确度与确定性。

### 8. 不足与局限

- **潜在信息丢失**：硬注意力剪枝（Top-K）可能丢失在低分辨率下缺乏显著性的关键线索，如微转移灶或孤立肿瘤细胞。虽然联合注意力校核能过滤伪阳性，但当前机制无法召回被错误丢弃的“有效区域”。

- **应用限制**：方法旨在辅助而非替代病理医生，论文明确强调仍需在多样化临床数据上进一步验证才能部署。此外，仅实现了分类任务，尚未拓展到分割或检测等更精细的下游应用。
