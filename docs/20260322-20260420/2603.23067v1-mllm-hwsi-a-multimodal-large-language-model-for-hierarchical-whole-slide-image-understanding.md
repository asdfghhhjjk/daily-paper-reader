## 摘要
全数字切片图像（Whole Slide Images, WSIs）具有层级结构，其诊断信息源自细胞形态、组织区域结构以及全局上下文。现有的计算病理学（Computational Pathology, CPath）多模态大型语言模型（Multimodal Large Language Models, MLLMs）通常将整个 WSI 压缩为一个单一嵌入，这限制了精细粒度的对齐表现，并忽略了病理学家如何在不同尺度上综合证据的过程。我们提出 \textbf{MLLM-HWSI}，一种层级化 WSI 级别的 MLLM，在四个不同尺度上——细胞作为单词、图块（patch）作为短语、区域作为句子、WSI 作为段落——将视觉特征与病理学语言对齐，以支持可解释的、基于证据的推理。MLLM-HWSI 将每个 WSI 分解为多尺度嵌入，并通过特定尺度的投影器联合施加 (i) 层级对比目标 和 (ii) 跨尺度一致性损失，从细胞到 WSI 保持语义连贯。我们计算与诊断相关的图块，并利用轻量级的 \textit{Cell-Cell Attention Fusion (CCAF)} Transformer 将分割的细胞嵌入聚合为每个图块的紧凑细胞 token。投影后的多尺度 token 与文本 token 融合后输入经指令微调的 LLM，用于开放性推理、视觉问答（VQA）、报告和描述生成等任务。MLLM-HWSI 经过三阶段训练，在 6 项 CPath 任务的 13 个 WSI 级基准上实现了新的 SOTA 结果。通过将语言与多尺度视觉证据对齐，MLLM-HWSI 提供准确、可解释的输出，反映诊断工作流程，提升整体 WSI 理解。代码可在 \href{https://github.com/BasitAlawode/HWSI-MLLM}{GitHub} 获取。

## Abstract
Whole Slide Images (WSIs) exhibit hierarchical structure, where diagnostic information emerges from cellular morphology, regional tissue organization, and global context. Existing Computational Pathology (CPath) Multimodal Large Language Models (MLLMs) typically compress an entire WSI into a single embedding, which hinders fine-grained grounding and ignores how pathologists synthesize evidence across different scales. We introduce \textbf{MLLM-HWSI}, a Hierarchical WSI-level MLLM that aligns visual features with pathology language at four distinct scales, cell as word, patch as phrase, region as sentence, and WSI as paragraph to support interpretable evidence-grounded reasoning. MLLM-HWSI decomposes each WSI into multi-scale embeddings with scale-specific projectors and jointly enforces (i) a hierarchical contrastive objective and (ii) a cross-scale consistency loss, preserving semantic coherence from cells to the WSI. We compute diagnostically relevant patches and aggregate segmented cell embeddings into a compact cellular token per-patch using a lightweight \textit{Cell-Cell Attention Fusion (CCAF)} transformer. The projected multi-scale tokens are fused with text tokens and fed to an instruction-tuned LLM for open-ended reasoning, VQA, report, and caption generation tasks. Trained in three stages, MLLM-HWSI achieves new SOTA results on 13 WSI-level benchmarks across six CPath tasks. By aligning language with multi-scale visual evidence, MLLM-HWSI provides accurate, interpretable outputs that mirror diagnostic workflows and advance holistic WSI understanding. Code is available at: \href{https://github.com/BasitAlawode/HWSI-MLLM}{GitHub}.

- **研究动机**：病理学诊断通常依赖“全数字切片图像（Whole Slide Image, WSI）”，它是多层次结构的巨像数据，包含从细胞形态到整体组织架构的丰富信息。  
- **现有问题**：当前计算病理学（Computational Pathology, CPath）领域的多模态大语言模型（MLLMs）大多将整张 WSI 压缩成单一嵌入，这种方式忽略了病理诊断中的层级结构与多尺度推理过程，导致：
  - 细粒度语义丧失；

  - 解释性较差，不符合真实诊断逻辑。
- **研究目标**：提出一种分层次、多尺度的 MLLM，使模型能够在 **细胞—图块（patch）—区域—整体 WSI 四个尺度**上进行语义对齐，实现符合病理学家工作流程的推理与生成。

## 二、方法论与技术框架

### 核心思想
- 将 WSI 的层级结构视为语言学的层级结构：  
  - **细胞 → 单词；图块 → 短语；区域 → 句子；WSI → 段落**。  
- 从细胞到全局逐级学习视觉与语言之间的对应关系，实现层级化视觉—语言（V-L）对齐。

### 模型结构组件
1. **Hierarchical Multi-Scale Encoder**
   - 分别提取四个层次的视觉特征：CellViT（细胞级）、CONCH（图块级）、HIPT（区域及 WSI级）。
   - 形成多尺度嵌入集合：{Cell, Patch, Region, WSI}。
2. **Semantic Patch Filtering (SPF)**
   - 通过与报告文本的余弦相似度筛除冗余图块，只保留诊断相关的部分。
3. **Cell–Cell Attention Fusion (CCAF)**
   - 利用轻量 ViT 对每个 patch 内的细胞嵌入做交叉注意力聚合，生成代表性细胞 token。
4. **Hierarchical V→L 投影器**
   - 为各层级定义独立的投影模块（Ac, Ap, Ar, AWSI）映射视觉嵌入到语言空间。
5. **多模态大型语言模型（LLM）**
   - 采用 Qwen2.5-7B-Instruct 作为后端 LLM，与视觉 token 融合后进行指令式推理、VQA、报告生成等任务。

### 优化目标
- **层级对比学习目标（Hierarchical Contrastive Alignment, Ls & LWSI）**：在各尺度上强化视觉—文本配对。  
- **跨尺度一致性损失（Cross-Scale Consistency, Lc）**：约束相邻层级特征保持语义平滑。  
- **综合损失公式**：  
  \( L_{HCA} = \sum(L_s + L_c) + L_{WSI} \)，保证局部与全局一致。

### 三阶段训练流程
1. **阶段1：层级跨模态对齐**（9,642 WSI–报告对）  
   - 优化多尺度 encoder 与文本编码器。
2. **阶段2：特征空间对齐**  
   - 训练投影矩阵以精调视觉—语言映射。
3. **阶段3：任务指令微调**（175,450 VQA 对）  
   - 联合调整投影及 LLM 权重以适应下游任务。
