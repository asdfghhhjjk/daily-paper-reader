## 摘要
传统的全切片图像（WSI）分析方法通常依赖于多实例学习（MIL）范式，该范式在高倍率下提取图像块级别的特征，并聚合这些特征用于切片级预测。然而，这种穷举式的图像块处理计算开销大，严重限制了WSI分析的效率和可扩展性。为解决这一挑战，我们提出了PathCTM（面向病理学的连续思维模型），该模型能够对十亿像素级别的WSI进行令牌高效的尺度空间连续推理。PathCTM将诊断推理形式化为动态序列信息追踪过程。它逐步从低倍率的全局观察过渡到高倍率的局部检查，并在收集到足够证据以有效限定决策不确定性时自适应地终止推理。具体而言，它采用条件计算实现动态尺度切换，并结合注意力引导的区域剪枝以及置信度感知的提前停止机制。大量实验表明，与标准基于MIL的方法相比，PathCTM减少了95.95%所需的图像块数量，并将推理时间缩短了约95.62%，同时AUC性能保持不变。代码已开源在https://github.com/JSGe-AI/PathCTM。

## Abstract
Traditional whole slide image (WSI) analysis methods typically rely on the multiple instance learning (MIL) paradigm, which extracts patch-level features at high magnification and aggregates them for slide-level prediction. However, such exhaustive patch-level processing is computationally expensive, severely limiting the efficiency and scalability of WSI analysis. To address this challenge, we propose PathCTM (a Pathology-oriented Continuous Thought Model) that enables token-efficient scale-space continuous reasoning for gigapixel WSIs. PathCTM formulates diagnostic inference as a dynamic sequential information pursuit. It progressively transitions from low-magnification global to high-magnification local inspection, and adaptively terminates inference when sufficient evidence is gathered to effectively bound decision uncertainty. Specifically, it uses conditional computation for dynamic scale switching with attention-guided region pruning, coupled with confidence-aware early stopping. Extensive experiments demonstrate that, compared with standard MIL-based methods, PathCTM reduces the number of required image patches by 95.95% and shortens inference time by approximately 95.62%, while maintaining AUC without degradation. Code is available at https://github.com/JSGe-AI/PathCTM.

## 2. 方法论

### 2.1 核心思想
- **“尺度思维”（Thinking in Scales）**：在连续推理模型中引入多尺度金字塔结构，将迭代推理与WSI的层次化分辨率对齐。
- **形式化**：将WSI分析视为最小化诊断条件熵的动态过程，通过状态连续性、注意力引导选择和置信度阈值实现自适应计算。

### 2.2 关键技术细节
- **尺度空间连续推理**：
  - 在每个尺度上执行n步内部时间步推理，维护预激活历史（FIFO队列）和同步输出历史，确保状态跨尺度连续。
  - 从低倍率（L）开始，若置信度不足则切换到更高分辨率（L-1），通过交叉融合将粗粒度全局信息与细粒度局部信息结合。
- **注意力引导区域剪枝**：
  - 利用当前推理轨迹中最高置信度时刻的注意力权重，选择Top-K区域（默认K=10）作为下一尺度的候选补丁。
  - 通过联合注意力校准机制抑制错误传播：在下一尺度对所选区域重新计算注意力分布，淘汰无效候选。
- **置信度感知早期停止**：
  - 置信度定义为 $C_{t,L}=1 - \frac{H(\hat{y}_{t,L})}{\log N}$ （N为类别数）。
  - 停止时间 $\tau = \min\{(t,L) | C_{t,L} \geq \delta\}$ ，默认阈值 $\delta=0.9$。
- **损失函数**：联合优化每个尺度的最小损失点和最大置信度点的交叉熵损失，引导模型既学会正确分类又学会识别何时确定。

## 6. 主要结论与发现
- **效率提升显著**：相比标准MIL方法，PathCTM平均减少95.95%的补丁数量，缩短95.62%的推理时间。

- **自适应推理深度**：简单任务（如BRACS-3）多在低倍率停止（平均183个补丁），复杂任务（如RCC分期、突变预测）倾向深入高倍率（平均250-408个补丁），与病理医生行为一致。
- **可解释性**：注意力图聚焦于上皮区域、导管结构、肿瘤-间质交界等关键诊断区域。

## 7. 优点
- **模仿人类认知**：动态“从粗到细”推理，与病理医生工作流高度对齐，可解释性强。
- **高效性**：通过条件计算和早期停止，实现计算量与实例困难度解耦。
- **兼容性**：可直接与现有病理基础模型（如CONCH, UNI）集成，无需额外训练。
- **理论支撑**：用Fano不等式为早期停止提供了信息论下界，用注意力作为影响代理提供了剪枝的理论依据。
